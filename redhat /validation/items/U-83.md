## [U-83] WebDAV 기능 비활성화

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | U-83 |
| 항목명 | WebDAV 기능 비활성화 |
| 위험도 | 상 (High) |
| 범주 | 서비스 관리 |

### 점검 목적

WebDAV는 HTTP를 통한 파일 관리(업로드·수정·삭제) 기능을 제공합니다. 활성화된 경우 공격자가 악성 파일을 서버에 업로드하거나 기존 파일을 수정할 수 있어, 웹쉘 업로드 등 심각한 보안 위협이 발생할 수 있습니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | WebDAV 모듈이 비활성화되고 Dav On 설정이 없는 경우 | HTTP를 통한 파일 업로드/수정 불가 |
| 취약 | dav_module이 로드되어 있거나 Dav On 설정이 존재하는 경우 | WebDAV를 통한 악성 파일 업로드 가능 |

## 베이스라인 기준

**안전(양호)**

- Apache: `dav_module`, `dav_fs_module` 비활성화(주석), `Dav On` 없음
- Nginx: `dav_methods` 설정 없음, DAV 모듈 미포함 빌드

**취약**

- `LoadModule dav_module` 활성 상태
- `Dav On` 설정 존재
