## [U-31] 흔적 숨기기 방지

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | U-31 |
| 항목명 | 흔적 숨기기 방지 |
| 위험도 | 중 (Medium) |
| 범주 | 파일 및 디렉토리 관리 |

### 점검 목적

공격자는 침입 후 `.bash_history` 파일 삭제, `/dev/null`로 링크, `HISTSIZE=0` 설정 등으로 자신의 행위 기록을 제거하려 합니다. 히스토리 파일 관련 설정을 강제하여 사용자 행위 로그가 보존되도록 해야 합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 히스토리 파일 크기가 적절히 설정되고, 심볼릭 링크 등 우회 설정이 없는 경우 | 사용자 명령 행위 로그 보존 |
| 취약 | HISTSIZE=0이거나 .bash_history가 /dev/null로 링크된 경우 | 사용자 행위 기록 삭제 가능 |

## 베이스라인 기준

**안전(양호)**

- `HISTSIZE=1000` 이상 (0 아님)
- `.bash_history`가 `/dev/null`로 링크되지 않음
- 히스토리 파일 삭제 또는 비활성화 없음

**취약**

- `HISTSIZE=0` 또는 `HISTFILE=/dev/null`
- `.bash_history` → `/dev/null` 심볼릭 링크
