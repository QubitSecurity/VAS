## [U-58] NFS 설정파일 접근 권한

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | U-58 |
| 항목명 | NFS 설정파일 접근 권한 |
| 위험도 | 중 (Medium) |
| 범주 | 보안 관리 |

### 점검 목적

`/etc/exports` 파일에 일반 사용자의 쓰기 권한이 있을 경우, 무단으로 NFS 공유 설정을 변경하여 민감한 디렉토리를 외부에 공개할 수 있습니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | /etc/exports 소유자가 root이고 권한이 644 이하인 경우 | 일반 사용자의 NFS 설정 변경 불가 |
| 취약 | /etc/exports 권한이 644를 초과하는 경우 | NFS 공유 설정 임의 변경 가능 |

## 베이스라인 기준

**안전(양호)**

- 소유자: `root`, 권한: `644` 또는 `640`

**취약**

- group/other write 권한 존재
