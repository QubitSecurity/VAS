## [U-52] Apache 불필요한 파일 제거

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | U-52 |
| 항목명 | Apache 불필요한 파일 제거 |
| 위험도 | 상 (High) |
| 범주 | 서비스 관리 |

### 점검 목적

Apache 설치 시 기본으로 생성되는 샘플 파일, 매뉴얼, 테스트 CGI 스크립트 등은 서비스 운영에 불필요하며 버전 정보 노출 또는 공격 경로로 활용될 수 있습니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 기본 설치 샘플 파일, 매뉴얼, 테스트 파일이 제거된 경우 | 불필요한 정보 노출 및 공격 경로 제거 |
| 취약 | Apache 기본 설치 파일이 그대로 존재하는 경우 | 버전 정보 노출, 테스트 스크립트 악용 가능 |

## 베이스라인 기준

**안전(양호)**

- `/var/www/html/` 기본 페이지 제거
- `manual.conf` 비활성화 또는 제거
- 기본 CGI 스크립트 제거

**취약**

- Apache 기본 `index.html`, `printenv`, `test-cgi` 등 존재
