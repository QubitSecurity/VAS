## [U-18] root 홈·PATH 디렉토리 권한 및 PATH 설정

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | U-18 |
| 항목명 | root 홈·PATH 디렉토리 권한 및 PATH 설정 |
| 위험도 | 상 (High) |
| 범주 | 파일 및 디렉토리 관리 |

### 점검 목적

root의 홈 디렉토리 또는 PATH에 포함된 디렉토리에 일반 사용자의 쓰기 권한이 부여될 경우, 악성 실행 파일을 해당 경로에 배치하여 root 실행을 유도할 수 있습니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | root 홈 디렉토리 권한이 750 이하이고, PATH 디렉토리에 타 사용자 쓰기 권한이 없는 경우 | Trojan horse 공격 방지 |
| 취약 | root 홈 디렉토리에 타 사용자 쓰기 권한이 있거나, PATH 디렉토리에 other write 권한이 있는 경우 | 악성 파일 삽입을 통한 root 권한 탈취 가능 |

## 베이스라인 기준

**안전(양호)**

- `/root` 권한: `550` 또는 `750` (other 권한 없음)
- PATH 내 모든 디렉토리 other write 권한 없음

**취약**

- `/root` 권한 `777` 또는 other write
- PATH 디렉토리에 other write 권한
