## [U-16] $HOME/.rhosts, hosts.equiv 사용 금지

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | U-16 |
| 항목명 | $HOME/.rhosts, hosts.equiv 사용 금지 |
| 위험도 | 상 (High) |
| 범주 | 계정 관리 |

### 점검 목적

`.rhosts` 및 `hosts.equiv` 파일은 패스워드 없이 지정된 호스트/계정으로부터의 접속을 허용합니다. 이는 IP 스푸핑 공격에 취약하며, 네트워크 레벨의 인증만으로 시스템 접근을 허용하는 매우 위험한 설정입니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | .rhosts 파일 및 /etc/hosts.equiv 파일이 존재하지 않거나 내용이 없는 경우 | r 계열 명령어의 패스워드 없는 인증 불가 |
| 취약 | .rhosts 또는 /etc/hosts.equiv 파일이 존재하고 허용 호스트/계정이 설정된 경우 | 패스워드 없이 원격 접근 허용, IP 스푸핑 취약 |

## 베이스라인 기준

**안전(양호)**

- `/etc/hosts.equiv` 파일 없음 또는 비어 있음
- 모든 홈 디렉토리에 `.rhosts` 파일 없음

**취약**

- `+ +` 또는 특정 호스트 허용 항목 존재
