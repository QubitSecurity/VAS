## [U-06] root 계정 su 제한

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | U-06 |
| 항목명 | root 계정 su 제한 |
| 위험도 | 상 (High) |
| 범주 | 계정 관리 |

### 점검 목적

`su` 명령어를 통한 root 전환을 모든 사용자에게 허용하면, 패스워드를 알고 있는 임의의 사용자가 root 권한으로 전환할 수 있습니다. `wheel` 그룹 멤버만 `su` 명령으로 root로 전환 가능하도록 제한하여 권한 남용을 방지해야 합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | wheel 그룹 멤버만 su 명령으로 root 전환 가능한 경우 | PAM pam_wheel.so 설정으로 su 명령어 실행 대상 제한 |
| 취약 | su 명령어 실행 대상에 제한이 없는 경우 | 모든 사용자가 root 패스워드만 알면 su로 root 전환 가능 |

## 베이스라인 기준

**안전(양호)**

- `/etc/pam.d/su`에 `auth required pam_wheel.so use_uid` 활성화
- `wheel` 그룹에 최소한의 관리자 계정만 포함

**취약**

- `/etc/pam.d/su`에 `pam_wheel.so` 설정이 주석 처리 또는 미설정
