## [U-51] Apache 상위 디렉토리 접근 금지

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | U-51 |
| 항목명 | Apache 상위 디렉토리 접근 금지 |
| 위험도 | 상 (High) |
| 범주 | 서비스 관리 |

### 점검 목적

디렉토리 트래버설(Directory Traversal) 취약점을 통해 공격자가 `../`를 사용하여 웹 루트 외부의 시스템 파일에 접근할 수 있습니다. `AllowOverride` 설정과 `Options` 지시어로 상위 디렉토리 접근을 제한해야 합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | AllowOverride None이고 루트 디렉토리 접근이 제한된 경우 | 상위 디렉토리 접근 및 .htaccess 오버라이드 차단 |
| 취약 | 디렉토리 트래버설 공격에 노출된 경우 | 시스템 파일 접근 가능 |

## 베이스라인 기준

**안전(양호)**

- 루트 `/` 디렉토리에 `Options None`, `AllowOverride None`, `Require all denied`
- 웹 루트 이외 디렉토리 접근 차단

**취약**

- `AllowOverride All` 또는 루트 디렉토리 접근 허용
