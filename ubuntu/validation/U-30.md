## [U-30] $HOME/.netrc 사용 금지

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | U-30 |
| 항목명 | $HOME/.netrc 사용 금지 |
| 위험도 | 상 (High) |
| 범주 | 파일 및 디렉토리 관리 |

### 점검 목적

`.netrc` 파일은 FTP, rexec 등에서 자동 로그인을 위해 호스트명, 계정, 패스워드를 평문으로 저장합니다. 이 파일이 존재할 경우 공격자가 파일 내용을 읽어 원격 시스템 접근 정보를 획득할 수 있습니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | .netrc 파일이 존재하지 않는 경우 | 평문 인증 정보 저장 없음 |
| 취약 | .netrc 파일이 존재하고 계정 및 패스워드가 저장된 경우 | 평문 인증 정보 노출 위험 |

## 베이스라인 기준

**안전(양호)**

- `.netrc` 파일 없음

**취약**

- `.netrc` 파일 존재 및 `password` 항목 평문 저장
