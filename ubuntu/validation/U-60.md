## [U-60] SNMP 설정파일 접근 권한

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | U-60 |
| 항목명 | SNMP 설정파일 접근 권한 |
| 위험도 | 중 (Medium) |
| 범주 | 보안 관리 |

### 점검 목적

SNMP 설정 파일에는 커뮤니티 스트링(인증 정보)이 포함됩니다. 일반 사용자가 이 파일을 읽을 수 있으면 커뮤니티 스트링을 획득하여 SNMP 서비스를 통해 시스템 정보를 수집하거나 설정을 변경할 수 있습니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | SNMP 설정 파일 소유자가 root이고 권한이 640 이하인 경우 | 커뮤니티 스트링 노출 차단 |
| 취약 | SNMP 설정 파일에 일반 사용자 읽기 권한이 있는 경우 | 커뮤니티 스트링 노출 |

## 베이스라인 기준

**안전(양호)**

- `/etc/snmp/snmpd.conf` 소유자: `root`, 권한: `640` 또는 `600`

**취약**

- `644`, `664`, `666` 등 other 읽기 가능
