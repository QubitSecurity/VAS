## [U-53] Apache 링크 사용 금지

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | U-53 |
| 항목명 | Apache 링크 사용 금지 |
| 위험도 | 상 (High) |
| 범주 | 서비스 관리 |

### 점검 목적

`FollowSymLinks` 옵션이 활성화되면 웹 서버가 심볼릭 링크를 따라 웹 루트 외부의 파일에 접근할 수 있습니다. 이를 통해 `/etc/passwd`, `/etc/shadow` 등 중요 시스템 파일이 노출될 수 있습니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | Options 지시어에서 FollowSymLinks 옵션이 제거된 경우 | 심볼릭 링크를 통한 웹 루트 외부 접근 차단 |
| 취약 | FollowSymLinks 옵션이 활성화된 경우 | 심볼릭 링크를 통한 시스템 파일 접근 가능 |

## 베이스라인 기준

**안전(양호)**

- `Options -FollowSymLinks` 또는 `Options None`
- 웹 루트에 심볼릭 링크 없음
- 설정 파일 경로: `/etc/apache2/apache2.conf`

**취약**

- `Options +FollowSymLinks` (웹 루트 외부 접근 가능)
