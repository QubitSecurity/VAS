## [U-24] /etc/rsyslog.conf 파일 소유자 및 권한 설정

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | U-24 |
| 항목명 | /etc/rsyslog.conf 파일 소유자 및 권한 설정 |
| 위험도 | 상 (High) |
| 범주 | 파일 및 디렉토리 관리 |

### 점검 목적

`/etc/rsyslog.conf`는 시스템 로그 수집 설정 파일입니다. 일반 사용자가 이 파일을 수정할 경우 로그 기록을 비활성화하거나 로그를 외부로 유출하는 설정을 추가할 수 있습니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 파일 소유자가 root이고 권한이 644 이하인 경우 | root만 쓰기, 일반 사용자 읽기만 가능 |
| 취약 | 소유자가 root가 아니거나 권한이 644를 초과하는 경우 | 로그 설정 변조 가능 |

## 베이스라인 기준

**안전(양호)**

- `/etc/rsyslog.conf` 소유자: `root`, 권한: `644` 또는 `640`

**취약**

- group/other write 권한 존재
