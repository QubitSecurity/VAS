## [U-09] 패스워드 최소 사용 기간 설정

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | U-09 |
| 항목명 | 패스워드 최소 사용 기간 설정 |
| 위험도 | 중 (Medium) |
| 범주 | 계정 관리 |

### 점검 목적

패스워드 최소 사용 기간을 설정하지 않으면 사용자가 패스워드 변경 직후 다시 이전 패스워드로 되돌릴 수 있어 패스워드 히스토리 정책이 무력화됩니다. 최소 1일 이상의 사용 기간을 설정해야 합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 패스워드 최소 사용 기간이 1일 이상으로 설정된 경우 | 패스워드 변경 후 즉시 재변경 불가 |
| 취약 | 패스워드 최소 사용 기간이 0일(즉시 변경 가능)로 설정된 경우 | 패스워드 히스토리 정책 우회 가능 |

## 베이스라인 기준

**안전(양호)**

- `PASS_MIN_DAYS 1` 이상

**취약**

- `PASS_MIN_DAYS 0`
