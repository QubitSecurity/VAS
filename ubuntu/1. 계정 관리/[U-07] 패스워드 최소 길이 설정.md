## [U-07] 패스워드 최소 길이 설정

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | U-07 |
| 항목명 | 패스워드 최소 길이 설정 |
| 위험도 | 중 (Medium) |
| 범주 | 계정 관리 |

### 점검 목적

짧은 패스워드는 무차별 대입 공격에 매우 취약합니다. 최소 8자리 이상의 패스워드 길이를 강제함으로써 패스워드 공간을 확장하고 공격 난이도를 높여야 합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 패스워드 최소 길이가 8자 이상으로 설정된 경우 | PAM pwquality 또는 login.defs에서 최소 길이 강제 |
| 취약 | 패스워드 최소 길이 설정이 없거나 8자 미만인 경우 | 짧은 패스워드 사용 허용, 무차별 대입 공격 용이 |

## 베이스라인 기준

**안전(양호)**

- `/etc/security/pwquality.conf`: `minlen = 8` 이상

**취약**

- `minlen` 미설정 또는 `minlen < 8`
- `/etc/login.defs`의 `PASS_MIN_LEN < 8`
