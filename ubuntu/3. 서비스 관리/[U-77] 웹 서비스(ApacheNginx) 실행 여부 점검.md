## [U-77] 웹 서비스(Apache/Nginx) 실행 여부 점검

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | U-77 |
| 항목명 | 웹 서비스(Apache/Nginx) 실행 여부 점검 |
| 위험도 | 상 (High) |
| 범주 | 서비스 관리 |

### 점검 목적

웹 서비스가 불필요하게 활성화되어 있으면 불필요한 공격 표면이 노출됩니다. 웹 서비스를 운영해야 하는 경우 최신 보안 패치 적용 및 하위 보안 설정(U-48~U-55, U-78~U-84)을 모두 점검해야 합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 웹 서비스가 비활성화되어 있거나, 운영 필요 시 보안 설정이 강화된 경우 | 불필요한 웹 서비스 노출 없음 |
| 취약 | 불필요한 웹 서비스가 활성화되어 있거나, 기본 설정으로 운영 중인 경우 | 웹 취약점 공격 표면 노출 |

## 베이스라인 기준

**안전(양호)**

- 웹 서비스 미운영 시: **`apache2`** 또는 `nginx` 서비스 비활성화 및 중지
- 웹 서비스 운영 시: U-48~U-55, U-78~U-84 항목 모두 안전 충족

**취약**

- 불필요한 `apache2` 또는 `nginx` 서비스 활성화 상태
- 웹 서비스 운영 중이나 하위 보안 설정 미적용
