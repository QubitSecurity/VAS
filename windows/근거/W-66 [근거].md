## [W-66] 불필요한 ODBC/OLE-DB 데이터 소스와 드라이버 제거

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-66 |
| 항목명 | 불필요한 ODBC/OLE-DB 데이터 소스와 드라이버 제거 |
| 위험도 | 중 (Medium) |
| 범주 | 접근 제어 |

### 점검 목적
불필요하게 설정된 ODBC/OLE-DB 데이터 소스에는 DB 접속 정보(서버 주소, 계정, 패스워드)가 포함될 수 있습니다. 공격자가 해당 설정에 접근하면 DB 서버에 직접 연결하거나 접속 정보를 탈취할 수 있습니다. 미사용 데이터 소스와 드라이버는 제거해야 합니다.

### 점검 기준
| 구분    | 기준                                            | 설명                                          |
| ----- | --------------------------------------------- | ------------------------------------------- |
| 안전    | 업무상 사용하지 않는 ODBC/OLE-DB 데이터 소스 및 드라이버가 제거된 경우 | 불필요한 DB 연결 경로 및 접속 정보 노출 위험이 감소한다           |
| 취약    | 불필요한 ODBC/OLE-DB 데이터 소스가 설정되어 있는 경우           | DB 접속 정보가 노출되거나 불필요한 DB 연결 경로가 공격에 악용될 수 있다 |


**베이스라인 기준**

 **양호** 
 
• **ODBC 데이터 원본 관리자** → **시스템 DSN** 탭에

  • **실제 업무용 데이터베이스 연결**만 존재
  
  • 샘플/테스트 DSN **없음**

 **취약**
 
• **시스템 DSN**에 아래 **불필요 항목 존재**:

  • `SQL Server (SAMPLE)`
  
  • `Northwind`
  
  • `pubs`
  
  • `AdventureWorks`
  
  • 기타 **샘플/테스트용** DSN


| 상태      | **정확한 조건**                                                          | **점검 방법**                                                                            |
| ------- | ------------------------------------------------------------------- | ------------------------------------------------------------------------------------ |
| **양호**  | `odbcad32.exe` → 시스템 DSN  <br>**업무 DB만 존재** (`PROD_DB`, `ERP_DB` 등) | `reg query "HKLM\SOFTWARE\ODBC\ODBC.INI\ODBC Data Sources"`  <br>**샘플 DSN 이름 없음** 확인 |
| ** 취약** | **시스템 DSN에 샘플/테스트 DSN 존재**                                          | `Northwind`, `pubs`, `SAMPLE` 등 **검출**                                               |
