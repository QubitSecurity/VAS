## [W-33] 백신 프로그램 업데이트

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-33 |
| 항목명 | 백신 프로그램 업데이트 |
| 위험도 | 상 (High) |
| 범주 | 패치 관리 |

### 점검 목적
백신 프로그램의 바이러스 정의 파일(DB)이 최신 상태가 아닌 경우, 신규 악성코드나 랜섬웨어를 탐지하지 못할 수 있습니다. 악성코드는 지속적으로 변종이 생성되므로 백신 엔진과 정의 파일은 항상 최신 상태로 유지해야 합니다.

### 점검 기준
| 구분    | 기준                                       | 설명                                  |
| ----- | ---------------------------------------- | ----------------------------------- |
| 안전    | 백신 프로그램의 엔진 및 정의 파일이 최신 상태로 유지되는 경우      | 최신 악성코드 및 랜섬웨어에 대한 탐지·차단이 가능하다      |
| 취약    | 백신 정의 파일이 오래되었거나 업데이트가 비활성화된 경우          | 신규 악성코드 탐지가 불가능하여 감염 위험이 높아진다       |


**베이스라인 기준**
──────────────────────────
**양호** 
• Windows Defender AntivirusSignatureLastUpdated ≥ 최근 7일 이내
• AntivirusEnabled = True (백신 활성화)
• RealTimeProtectionEnabled = True (실시간 보호 활성화)
• 자동 업데이트 기능 활성화 상태

**취약**
• AntivirusSignatureLastUpdated < 7일 경과
• 자동 업데이트 미활성화
• 실시간 보호 기능 비활성화
──────────────────────────
