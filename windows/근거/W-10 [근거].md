## [W-10] IIS 서비스 구동 점검

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-10 |
| 항목명 | IIS 서비스 구동 점검 |
| 위험도 | 상 (High) |
| 범주 | IIS 관리 |

### 점검 목적
IIS(Internet Information Services)는 웹 서비스를 제공하는 서버 소프트웨어로, 불필요하게 구동 중인 경우 외부 공격의 주요 진입점이 됩니다. 웹 서비스가 필요하지 않은 서버에서 IIS가 실행 중이라면 즉시 중지해야 하며, 필요한 경우라도 보안 설정을 강화해야 합니다.

### 점검 기준
| 구분    | 기준                                                 | 설명                               |
| ----- | -------------------------------------------------- | -------------------------------- |
| 안전    | IIS 서비스가 비활성화되어 있거나, 업무상 필요에 의해 운영되며 보안 설정이 적용된 경우 | 불필요한 웹 서비스 노출이 없어 외부 공격 표면이 감소한다 |
| 취약    | 업무상 불필요한 IIS 서비스가 실행 중인 경우                         | 불필요한 웹 서버가 외부 공격의 진입점으로 악용될 수 있다 |


## **베이스라인 기준**

 **안전(양호)** 
 
• services.msc에서 다음 서비스 **"중지"**이거나 **존재하지 않음**:
  - W3SVC (World Wide Web Publishing)
  - IISADMIN (IIS Admin Service)  
  - WAS (Windows Process Activation Service)

 **취약**
 
• 위 서비스 중 **하나 이상 "실행 중(Running)"**
