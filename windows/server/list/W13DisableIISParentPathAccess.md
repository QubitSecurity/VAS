KISA(한국인터넷진흥원)의 주요정보통신기반시설 기술적 취약점 분석·평가 가이드에 있는 **W-13 (W13DisableIISParentPathAccess)** 항목은 **"IIS 상위 디렉터리 접근 금지"**를 의미합니다.

한마디로 요약하면 **"웹 사이트 내에서 `../` (점점 슬래시) 같은 명령어를 사용해, 허용된 폴더 바깥쪽(상위)으로 나가는 것을 막아라"**는 뜻입니다.

상세한 의미와 보안 위협, 조치 방법은 다음과 같습니다.

---

### 1. 핵심 의미: `..` (부모 경로) 차단

컴퓨터에서 **`..` (점 두 개)**는 "현재 폴더의 바로 위(부모) 폴더"를 의미합니다.

* **설정 켜짐 (취약 - 접근 허용):** 웹 소스코드(ASP 등)에서 `../../Windows/System32/cmd.exe`와 같은 경로를 입력하면, 웹 루트(홈페이지 폴더)를 벗어나 윈도우 시스템 폴더까지 접근할 수 있습니다.
* **설정 꺼짐 (양호 - 접근 차단):** 웹 소스코드에서 `..`을 사용하는 순간 IIS가 차단합니다. 오직 현재 폴더와 그 하위 폴더만 접근할 수 있습니다.

### 2. 점검 목적 (Why?)

이 설정이 켜져 있으면 **상위 디렉터리 접근 공격(Directory Traversal)**에 노출될 수 있습니다.

1. **시스템 파일 접근:** 해커가 웹 서버의 파일 구조를 파악한 뒤, 웹 루트를 빠져나와 서버의 중요한 설정 파일(`boot.ini`, `system.ini` 등)이나 시스템 명령어(`cmd.exe`)에 접근하여 실행할 수 있습니다.
2. **소스코드 노출:** 다른 웹 사이트의 소스코드나 데이터베이스 파일 등 접근해서는 안 되는 민감한 파일에 접근할 수 있습니다.

### 3. 점검 기준 (Pass/Fail)

KISA 가이드라인에 따른 판단 기준은 다음과 같습니다.

* **양호 (Safe):** IIS 설정에서 **"부모 경로 사용(Enable Parent Paths)"** 옵션이 **False(사용 안 함)**로 설정된 경우
* **취약 (Vulnerable):** 해당 옵션이 **True(사용)**로 설정된 경우

### 4. 설정 확인 및 조치 방법 (IIS 관리자)

이 설정은 IIS의 **ASP (Active Server Pages)** 기능 설정에 위치해 있습니다.

1. **IIS 관리자 실행:** `Win + R` > `inetmgr` 입력
2. **사이트 선택:** 설정할 웹 사이트 클릭
3. **ASP 기능 열기:**
* 중앙의 IIS 섹션에서 **[ASP]** 아이콘을 찾아 더블 클릭합니다.


4. **동작(Behavior) 섹션 확인:**
* **"부모 경로 사용 (Enable Parent Paths)"** 항목을 찾습니다.


5. **조치:**
* 이 값을 **`False`**로 변경합니다.
* 우측 작업 패널에서 **[적용]**을 클릭합니다.



### 💡 주의사항: 구형(Legacy) 웹 사이트

만약 회사의 웹 사이트가 아주 오래된 **Classic ASP**로 만들어졌고, 개발자가 코드를 짤 때 `include file="../config/db.inc"` 처럼 습관적으로 상대 경로(`..`)를 많이 썼다면?

* 이 설정을 `False`로 바꾸는 순간 **웹 사이트 에러(500 Error)**가 발생할 수 있습니다.
* **해결책:**
1. (권장) 소스코드를 수정하여 `..` 대신 **가상 경로(Virtual Path, `/`)**를 사용하도록 변경합니다. (예: `/config/db.inc`)
2. (임시) 코드 수정이 불가능하다면 부득이하게 `True`로 두어야 하지만, 이는 보안 취약점으로 남게 되므로 별도의 보안 장비(WAF) 등으로 상위 접근 시도를 차단해야 합니다.



### 요약

> **W-13 항목은 "웹 사이트가 자기 집(홈 디렉터리) 밖으로 나가지 못하게 문을 잠가라"는 뜻입니다.**

`..` 경로를 허용하면 해커가 웹 서버의 안방(시스템 폴더)까지 드나들 수 있기 때문입니다.
