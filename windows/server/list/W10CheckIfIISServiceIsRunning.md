KISA(한국인터넷진흥원)의 주요정보통신기반시설 기술적 취약점 분석·평가 가이드에 있는 **W-10 (W10CheckIfIISServiceIsRunning)** 항목은 **"IIS 서비스 구동 여부 점검"**을 의미합니다.

한마디로 요약하면 **"이 서버가 웹 사이트를 운영하는 '웹 서버'가 아니라면, IIS(웹 서버 프로그램)를 끄거나 삭제하라"**는 뜻입니다.

상세한 의미와 보안 목적은 다음과 같습니다.

---

### 1. 핵심 의미: 불필요한 서비스 제거

**IIS (Internet Information Services)**는 마이크로소프트 윈도우에서 웹 사이트를 만들고 호스팅할 수 있게 해주는 **웹 서버 소프트웨어**입니다.

* **웹 서버라면:** 당연히 IIS가 켜져 있어야 합니다. (이 경우 W-10은 '점검' 단계에서 넘어가고, IIS 세부 보안 설정인 W-06~W-19 등을 확인하게 됩니다.)
* **웹 서버가 아니라면 (DB서버, 파일서버, AD서버 등):** IIS가 켜져 있을 이유가 전혀 없습니다. 그런데 윈도우 설치 시 기본적으로 깔려있거나, 실수로 설치되어 돌아가고 있다면 이것이 바로 **취약점**입니다.

### 2. 점검 목적 (Why?)

"안 쓰는 기능 좀 켜져 있으면 어떤가요?"라고 생각할 수 있지만, 보안에는 **'최소 권한 및 최소 기능의 원칙'**이 적용됩니다.

1. **공격 표면(Attack Surface) 최소화:**
* IIS 서비스가 켜져 있다는 것은 외부에서 접속 가능한 **80번(HTTP), 443번(HTTPS) 포트**가 열려 있다는 뜻입니다.
* 해커는 이 열린 문을 통해 웹 취약점 공격(SQL 인젝션, 파일 업로드 공격 등)을 시도하여, 웹 서버를 장악한 뒤 뒤쪽의 DB나 내부망으로 침투할 수 있습니다.


2. **불필요한 리소스 낭비 방지:** 사용하지 않는 서비스가 시스템 자원(CPU, 메모리)을 점유하는 것을 막습니다.

### 3. 점검 기준 (Pass/Fail)

KISA 가이드라인에 따른 판단 기준은 **서버의 용도**에 따라 달라집니다.

* **양호 (Safe):**
* IIS 서비스가 **중지(Stopped)**되어 있거나, 아예 **설치되지 않은** 경우 (웹 서버가 아닌 경우)
* 또는, 웹 서버 용도로 **필요에 의해** IIS를 사용 중인 경우 (단, 이 경우엔 불필요한 샘플 파일 제거 등 추가 보안 설정이 되어 있어야 함)


* **취약 (Vulnerable):**
* 웹 서비스를 하지 않는 서버인데 IIS 서비스가 **"실행 중(Running)"**인 경우



### 4. 확인 및 조치 방법

이 서버가 웹 서버가 아니라면, 다음 방법으로 서비스를 내리거나 제거해야 합니다.

#### A. 서비스 중지 (일시적 조치)

1. **실행 창 열기:** `Win + R` 누르고 `services.msc` 입력
2. **서비스 찾기:**
* **"World Wide Web Publishing Service"** 항목을 찾습니다. (이게 IIS의 핵심 서비스입니다.)


3. **조치:**
* 더블 클릭 후 시작 유형을 **[사용 안 함]**으로 변경하고, 서비스 상태를 **[중지]**로 설정합니다.



#### B. 역할 제거 (권장 - 영구적 조치)

아예 윈도우에서 IIS 기능을 삭제하는 것이 가장 안전합니다.

1. **서버 관리자(Server Manager)** 실행
2. **관리(Manage)** > **역할 및 기능 제거(Remove Roles and Features)** 클릭
3. **"웹 서버(IIS)"** 체크 박스를 해제하고 제거 진행

### 요약

> **W-10 항목은 "웹 서버도 아니면서 왜 대문(80번 포트)을 열어놓고 있느냐?"라고 묻는 항목입니다.**
> DB서버나 백업 서버 등 웹 서비스가 필요 없는 장비라면, 보안을 위해 IIS를 반드시 삭제하거나 꺼야 합니다.

혹시 현재 점검 중인 서버가 웹 서버인가요, 아니면 다른 용도의 서버인가요? 용도에 따라 조치 방향이 달라집니다.
