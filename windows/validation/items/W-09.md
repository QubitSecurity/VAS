## [W-09] 불필요한 서비스 제거

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-09 |
| 항목명 | 불필요한 서비스 제거 |
| 위험도 | 상 (High) |
| 범주 | 서비스 관리 |

### 점검 목적
실행 중인 서비스는 각각 독립적인 공격 표면(Attack Surface)이 됩니다. 업무상 불필요한 서비스가 실행 중인 경우, 해당 서비스의 취약점이 악용되거나 서비스 계정을 통한 권한 상승 경로로 활용될 수 있습니다. 필요하지 않은 서비스는 중지하고 시작 유형을 비활성화로 설정해야 합니다.

### 점검 기준
| 구분    | 기준                                             | 설명                                         |
| ----- | ---------------------------------------------- | ------------------------------------------ |
| 안전    | 업무상 불필요한 서비스가 중지 및 비활성화된 경우                    | 공격 표면이 최소화되어 서비스 취약점을 통한 침해 위험이 감소한다       |
| 취약    | 불필요한 서비스가 실행 중이거나 자동 시작으로 설정된 경우               | 불필요한 서비스의 취약점이 공격자에게 악용될 수 있다              |
