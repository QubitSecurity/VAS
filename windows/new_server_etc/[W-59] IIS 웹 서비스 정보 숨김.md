## [W-59] IIS 웹 서비스 정보 숨김

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-59 |
| 항목명 | IIS 웹 서비스 정보 숨김 |
| 위험도 | 중 (Medium) |
| 범주 | 접근 제어 |

### 점검 목적
HTTP 응답 헤더에 IIS 버전, ASP.NET 버전 등 서버 정보가 노출되는 경우, 공격자는 해당 버전의 알려진 취약점을 검색하여 표적 공격을 수행할 수 있습니다. 서버 정보는 응답 헤더에서 제거하거나 최소화하여 공격자의 정보 수집을 차단해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | HTTP 응답 헤더에서 Server, X-Powered-By 등 서버 정보가 제거된 경우 | 공격자가 서버 버전 정보를 수집하여 표적 공격을 수행하기 어려워진다 |
| 취약 | HTTP 응답 헤더에 IIS 버전, ASP.NET 버전 등 상세 정보가 노출되는 경우 | 공격자가 해당 버전의 알려진 취약점을 즉시 파악하여 표적 공격이 가능하다 |
