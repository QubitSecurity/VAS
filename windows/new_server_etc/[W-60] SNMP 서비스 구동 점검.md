## [W-60] SNMP 서비스 구동 점검(재검토)

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-60 |
| 항목명 | SNMP 서비스 구동 점검 |
| 위험도 | 중 (Medium) |
| 범주 | 접근 제어 |

### 점검 목적
SNMP(Simple Network Management Protocol)는 네트워크 장비 및 서버 모니터링에 사용되는 프로토콜입니다. 업무상 불필요하게 실행 중인 경우 또는 기본 커뮤니티 스트링(public/private)을 사용하는 경우, 공격자가 SNMP를 통해 시스템 정보를 수집하거나 설정을 변경할 수 있습니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | SNMP 서비스가 비활성화되었거나 업무상 필요한 경우 보안 설정이 적용된 경우 | 불필요한 SNMP 정보 노출 및 설정 변경 위험이 차단된다 |
| 취약 | 불필요한 SNMP 서비스가 실행 중인 경우 | SNMP를 통한 시스템 정보 수집 및 설정 변조 위험에 노출된다 |

### 참고 문서
- SNMP: SNMP(Simple Network Management Protocol)는 MIB(Management Information 
Base)에 기반을 둔 네트워크망을 관리하기 위한 목적으로 만들어진 프로토콜로, 간단한 명령으로 
원격 시스템의 CPU 정보에서부터, 인터페이스별 트래픽량 등 여러 가지 정보를 확인 가능
※관련 점검항목: W-30(중), W-31(중)
