## [W-28] FTP 접근 제어 설정

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-28 |
| 항목명 | FTP 접근 제어 설정 |
| 위험도 | 상 (High) |
| 범주 | 네트워크 서비스 |

### 점검 목적
FTP 서비스가 모든 IP로부터 접근 가능한 경우, 외부 공격자가 무차별 대입 공격이나 익명 접속 시도를 제한 없이 수행할 수 있습니다. 업무상 허용된 IP 대역에서만 접근하도록 IP 기반 접근 제어를 설정하여 공격 표면을 최소화해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 허용된 IP 대역에서만 FTP 접근이 가능하도록 설정된 경우 | 불특정 외부 접근이 차단되어 공격 시도 자체가 제한된다 |
| 취약 | 모든 IP에서 FTP 접근이 허용된 경우 | 외부 공격자가 제한 없이 FTP 접근을 시도할 수 있다 |
