## [W-72] DOS 공격 방어 레지스트리 설정

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-72 |
| 항목명 | DOS 공격 방어 레지스트리 설정 |
| 위험도 | 중 (Medium) |
| 범주 | 로그 및 감사 |

### 점검 목적
TCP SYN Flood 등의 DoS 공격을 방어하기 위해 연결 대기열 크기, 재전송 제한 등을 조정해야 합니다. 기본 설정으로는 대규모 동시 접속 요청에 취약하므로 적절한 레지스트리 설정을 통해 DoS 공격에 대한 내성을 강화해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | DoS 방어 관련 레지스트리 값이 적절히 설정된 경우 | 대규모 동시 접속 요청에 대한 내성이 강화되어 DoS 공격에 대응할 수 있다 |
| 취약 | DoS 방어 레지스트리 설정이 기본값으로 유지되는 경우 | 기본 설정으로는 SYN Flood 등 DoS 공격에 취약하다 |

### 참고 문서
- DoS(서비스 거부 공격): 네트워크 사용자가 컴퓨터나 컴퓨터의 특정 서비스를 사용할 수 없도록 만
들기 위한 네트워크 공격 [W-54]
