## [W-14] IIS 불필요한 파일 제거

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-14 |
| 항목명 | IIS 불필요한 파일 제거 |
| 위험도 | 상 (High) |
| 범주 | IIS 관리 |

### 점검 목적
IIS 설치 시 기본으로 생성되는 샘플 파일, 관리 스크립트, 기본 웹 페이지 등은 알려진 경로에 위치하며 취약점이 공개된 경우가 많습니다. 이러한 파일이 제거되지 않으면 공격자가 해당 경로를 직접 호출하여 정보 수집이나 공격에 활용할 수 있습니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | IIS 기본 샘플 파일, 관리 스크립트 등 불필요한 파일이 제거된 경우 | 알려진 취약 경로가 제거되어 공격자의 초기 접근 시도가 차단된다 |
| 취약 | IIS 기본 설치 파일이 그대로 잔존하는 경우 | 알려진 경로의 취약한 파일을 통해 정보 수집 및 공격이 가능하다 |
| 확인 필요 | 파일 존재 여부는 확인되나 업무상 사용 여부를 자동으로 판단할 수 없는 경우 | 담당자가 해당 파일의 업무 필요성을 수동으로 검토해야 한다 |
