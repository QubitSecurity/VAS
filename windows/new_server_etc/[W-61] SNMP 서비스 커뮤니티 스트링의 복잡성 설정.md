## [W-61] SNMP 서비스 커뮤니티 스트링의 복잡성 설정

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-61 |
| 항목명 | SNMP 서비스 커뮤니티 스트링의 복잡성 설정 |
| 위험도 | 중 (Medium) |
| 범주 | 접근 제어 |

### 점검 목적
SNMP 커뮤니티 스트링은 SNMP 접근을 위한 패스워드 역할을 합니다. 기본값인 `public`(읽기) 또는 `private`(읽기/쓰기)를 사용하는 경우, 공격자가 이미 알려진 기본값으로 즉시 접근하여 시스템 정보를 수집하거나 설정을 변경할 수 있습니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 커뮤니티 스트링이 기본값이 아닌 복잡한 문자열로 변경된 경우 | 기본값을 통한 SNMP 무단 접근이 차단된다 |
| 취약 | 커뮤니티 스트링이 public, private 등 기본값으로 설정된 경우 | 공격자가 기본값으로 즉시 SNMP에 접근하여 시스템 정보를 탈취하거나 설정을 변경할 수 있다 |

### 참고 문서
- 관련 점검항목: W-29(중), W-31(중) [W-30]
