## [W-66] 불필요한 ODBC/OLE-DB 데이터 소스와 드라이버 제거

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-66 |
| 항목명 | 불필요한 ODBC/OLE-DB 데이터 소스와 드라이버 제거 |
| 위험도 | 중 (Medium) |
| 범주 | 접근 제어 |

### 점검 목적
불필요하게 설정된 ODBC/OLE-DB 데이터 소스에는 DB 접속 정보(서버 주소, 계정, 패스워드)가 포함될 수 있습니다. 공격자가 해당 설정에 접근하면 DB 서버에 직접 연결하거나 접속 정보를 탈취할 수 있습니다. 미사용 데이터 소스와 드라이버는 제거해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 업무상 사용하지 않는 ODBC/OLE-DB 데이터 소스 및 드라이버가 제거된 경우 | 불필요한 DB 연결 경로 및 접속 정보 노출 위험이 감소한다 |
| 취약 | 불필요한 ODBC/OLE-DB 데이터 소스가 설정되어 있는 경우 | DB 접속 정보가 노출되거나 불필요한 DB 연결 경로가 공격에 악용될 수 있다 |
| 확인 필요 | 데이터 소스 목록은 확인되나 업무상 필요 여부를 자동으로 판단할 수 없는 경우 | 담당자가 데이터 소스 목록을 직접 검토하여 불필요 항목을 식별해야 한다 |

### 참고 문서
- 특정 샘플 응용프로그램은 샘플 데이터베이스를 위해 ODBC 데이터 소스를 설치하거나 불필요한 
ODBC/OLE-DB 데이터베이스 드라이버를 설치하므로 불필요한 데이터 소스나 드라이버는 
ODBC 데이터 소스 관리자 도구를 이용해서 제거하는 것이 바람직함 [W-35]
