## [W-19] IIS 가상 디렉토리 삭제

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-19 |
| 항목명 | IIS 가상 디렉토리 삭제 |
| 위험도 | 상 (High) |
| 범주 | IIS 관리 |

### 점검 목적
가상 디렉토리는 웹 루트 외부의 파일 시스템 경로를 웹에서 접근 가능하도록 매핑하는 기능입니다. 업무상 필요 없는 가상 디렉토리가 설정된 경우 해당 경로의 파일들이 웹을 통해 외부에 노출될 수 있으며, 공격자의 파일 탈취 및 정보 수집에 악용될 수 있습니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 불필요한 가상 디렉토리가 모두 제거된 경우 | 웹을 통한 불필요한 파일 시스템 접근 경로가 차단된다 |
| 취약 | 업무상 불필요한 가상 디렉토리가 설정되어 있는 경우 | 해당 경로의 파일이 웹을 통해 외부에 노출될 수 있다 |
| 확인 필요 | 가상 디렉토리 목록은 확인되나 업무 필요 여부를 자동으로 판단할 수 없는 경우 | 담당자가 가상 디렉토리 목록을 검토하여 불필요 항목을 수동으로 식별해야 한다 |

### 참고 문서
- 가상 디렉터리: 물리적으로 홈 디렉터리와 다른 위치에 있거나 홈 디렉터리에 종속된 디렉터리가 아니어도 웹 브라우저를 통해 사용자가 접속할 때 웹사이트의 하위 디렉터리로 인식되도록 하는 기법 [WEB-17]
