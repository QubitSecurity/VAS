## [W-73] 사용자가 프린터 드라이버를 설치할 수 없게 함

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-73 |
| 항목명 | 사용자가 프린터 드라이버를 설치할 수 없게 함 |
| 위험도 | 중 (Medium) |
| 범주 | 접근 제어 |

### 점검 목적
일반 사용자가 프린터 드라이버를 설치할 수 있는 경우, 악성 프린터 드라이버를 통해 권한 상승 공격이나 시스템 감염이 가능합니다. 프린터 드라이버 설치를 Administrators에게만 허용하여 악성 드라이버 설치 위험을 차단해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 프린터 드라이버 설치 권한이 Administrators에게만 부여된 경우 | 악성 프린터 드라이버를 통한 권한 상승 및 시스템 감염이 차단된다 |
| 취약 | 일반 사용자가 프린터 드라이버를 설치할 수 있는 경우 | 악성 드라이버를 통해 권한 상승 및 시스템 감염 공격이 가능하다 |
