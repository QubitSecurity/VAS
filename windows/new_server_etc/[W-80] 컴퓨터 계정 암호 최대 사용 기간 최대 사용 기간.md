## [W-80] 컴퓨터 계정 암호 최대 사용 기간 최대 사용 기간

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-80 |
| 항목명 | 컴퓨터 계정 암호 최대 사용 기간 |
| 위험도 | 중 (Medium) |
| 범주 | 계정 관리 |

### 점검 목적
도메인 환경에서 컴퓨터 계정은 도메인 컨트롤러와 인증을 위해 사용하는 서비스 계정입니다. 컴퓨터 계정 패스워드가 장기간 변경되지 않으면 탈취된 계정으로 도메인 자원에 무기한 접근이 가능합니다. 주기적인 컴퓨터 계정 패스워드 변경을 강제해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 컴퓨터 계정 패스워드 최대 사용 기간이 30일 이하로 설정된 경우 | 컴퓨터 계정 패스워드 주기 변경으로 탈취된 계정의 유효 기간이 제한된다 |
| 취약 | 컴퓨터 계정 패스워드 변경 주기가 너무 길거나 설정되지 않은 경우 | 탈취된 컴퓨터 계정으로 장기간 도메인 자원에 접근할 수 있다 |
