## [W-12] IIS CGI 실행 제한

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-12 |
| 항목명 | IIS CGI 실행 제한 |
| 위험도 | 상 (High) |
| 범주 | IIS 관리 |

### 점검 목적
CGI(Common Gateway Interface)는 웹 서버에서 외부 프로그램을 실행하는 인터페이스입니다. CGI 실행 권한이 불필요하게 허용된 경우, 공격자가 악의적인 CGI 스크립트를 업로드하거나 실행하여 서버에서 임의의 명령을 수행하는 원격 코드 실행(RCE) 공격으로 이어질 수 있습니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 업무상 불필요한 CGI 실행이 비활성화된 경우 | CGI를 통한 임의 명령 실행 경로가 차단된다 |
| 취약 | CGI 실행이 불필요하게 활성화된 경우 | 악성 CGI 스크립트 실행을 통한 원격 코드 실행 위험이 있다 |

### 참고 문서
- CGI(Common Gateway Interface): 사용자가 서버로 보낸 데이터를 서버에서 작동 중인 데이터 처리프로그램에 전달하고, 여기에서 처리된 데이터를 다시 서버로 되돌려 보내는 등의 일을 하는 표준 인터페이스 규격 [WEB-05]
