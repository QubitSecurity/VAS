## [W-46] Everyone 사용권한을 익명 사용자에 적용 해제

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-46 |
| 항목명 | Everyone 사용권한을 익명 사용자에 적용 해제 |
| 위험도 | 중 (Medium) |
| 범주 | 패스워드 정책 |

### 점검 목적
Windows에서 Everyone 그룹에 익명 사용자(Anonymous)가 포함될 경우, 인증되지 않은 외부 사용자도 Everyone 권한이 부여된 모든 자원에 접근할 수 있게 됩니다. 익명 사용자를 Everyone 그룹에서 제외하여 인증되지 않은 접근을 원천 차단해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | Everyone 그룹에 익명 사용자가 포함되지 않는 경우 | 인증되지 않은 사용자가 Everyone 권한 자원에 접근할 수 없다 |
| 취약 | Everyone 그룹에 익명 사용자가 포함된 경우 | 익명 사용자도 Everyone 권한이 부여된 공유 자원에 접근 가능하다 |
