## [W-40] 원격 시스템에서 강제로 시스템 종료

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-40 |
| 항목명 | 원격 시스템에서 강제로 시스템 종료 |
| 위험도 | 상 (High) |
| 범주 | 보안 설정 |

### 점검 목적
"원격에서 강제로 시스템 종료" 권한이 일반 사용자나 불필요한 그룹에 부여된 경우, 원격 접근 권한을 획득한 공격자가 서버를 강제 종료하여 서비스 중단(DoS)을 유발할 수 있습니다. 이 권한은 Administrators 그룹에만 부여해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 원격 강제 종료 권한이 Administrators 그룹에만 부여된 경우 | 권한 없는 사용자에 의한 원격 서비스 중단이 차단된다 |
| 취약 | 원격 강제 종료 권한이 일반 사용자 또는 불필요한 그룹에 부여된 경우 | 권한을 가진 계정 탈취 시 원격에서 서버를 강제 종료할 수 있다 |
