## [W-30] RDS(Remote Data Services) 제거

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-30 |
| 항목명 | RDS(Remote Data Services) 제거 |
| 위험도 | 상 (High) |
| 범주 | 서비스 관리 |

### 점검 목적
RDS(Remote Data Services)는 IIS와 함께 설치되는 구형 데이터 접근 컴포넌트로, 원격에서 DB에 접근하는 기능을 제공합니다. 알려진 취약점(MS99-025 등)을 통해 공격자가 원격에서 임의 명령을 실행하거나 DB에 무단 접근할 수 있어 사용 여부와 관계없이 제거해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | RDS 컴포넌트가 제거되었거나 비활성화된 경우 | 알려진 RDS 취약점을 통한 원격 공격 경로가 차단된다 |
| 취약 | RDS 컴포넌트가 설치되어 있는 경우 | 원격 코드 실행 및 DB 무단 접근 취약점에 노출될 수 있다 |
