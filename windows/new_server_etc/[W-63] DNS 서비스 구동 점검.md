## [W-63] DNS 서비스 구동 점검

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-63 |
| 항목명 | DNS 서비스 구동 점검 |
| 위험도 | 중 (Medium) |
| 범주 | 접근 제어 |

### 점검 목적
DNS 서버 역할이 없는 시스템에서 DNS 서비스가 실행 중인 경우, 불필요한 공격 표면이 생성됩니다. DNS 서비스의 취약점이 악용되거나 DNS 캐시 포이즈닝 공격의 대상이 될 수 있으므로 업무상 불필요한 DNS 서비스는 즉시 중지해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | DNS 서비스가 비활성화되어 있거나 DNS 서버 역할로 운영되며 보안 설정이 적용된 경우 | 불필요한 DNS 서비스 노출이 없어 관련 공격 위험이 감소한다 |
| 취약 | 업무상 불필요한 DNS 서비스가 실행 중인 경우 | DNS 취약점을 통한 공격 및 캐시 포이즈닝 위험에 노출된다 |

### 참고 문서
- 동적 업데이트: DNS 정보에 변경 사항이 있을 때마다 DNS 클라이언트 컴퓨터가 자신의 리소스 레
코드(zone 파일)를 DNS 서버에 자동으로 업데이트하는 기능으로 영역 레코드 수동 관리 작업을 줄
일 수 있음 [W-32]
