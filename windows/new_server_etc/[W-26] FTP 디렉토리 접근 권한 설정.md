## [W-26] FTP 디렉토리 접근 권한 설정

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-26 |
| 항목명 | FTP 디렉토리 접근 권한 설정 |
| 위험도 | 상 (High) |
| 범주 | 네트워크 서비스 |

### 점검 목적
FTP 루트 디렉토리 또는 하위 디렉토리에 과도한 쓰기 권한이 부여된 경우, 인증된 사용자뿐 아니라 익명 접속자도 악성 파일을 업로드하거나 서버 파일을 변조·삭제할 수 있습니다. FTP 디렉토리는 업무상 필요한 최소 권한만 부여해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | FTP 디렉토리에 필요한 최소 권한만 부여된 경우 | 파일 무단 업로드, 변조, 삭제 위험이 감소한다 |
| 취약 | FTP 디렉토리에 Everyone 쓰기 권한 또는 과도한 권한이 부여된 경우 | 악성 파일 업로드 및 서버 파일 무단 변조가 가능하다 |
