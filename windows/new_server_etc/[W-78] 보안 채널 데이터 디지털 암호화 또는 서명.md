## [W-78] 보안 채널 데이터 디지털 암호화 또는 서명

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-78 |
| 항목명 | 보안 채널 데이터 디지털 암호화 또는 서명 |
| 위험도 | 중 (Medium) |
| 범주 | 네트워크 인증 |

### 점검 목적
도메인 컨트롤러와 클라이언트 간 보안 채널 통신이 암호화 또는 서명되지 않은 경우, 네트워크 스니핑을 통해 도메인 자격증명, 정책 정보 등이 노출될 수 있습니다. 보안 채널 통신에 디지털 서명과 암호화를 강제하여 통신 보안성을 확보해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 보안 채널 데이터에 디지털 서명 및 암호화가 모두 활성화된 경우 | 도메인 통신 데이터가 네트워크 스니핑에 노출되지 않는다 |
| 취약 | 보안 채널 데이터 디지털 서명 또는 암호화가 비활성화된 경우 | 도메인 자격증명 및 정책 정보가 평문으로 전송되어 노출될 수 있다 |
