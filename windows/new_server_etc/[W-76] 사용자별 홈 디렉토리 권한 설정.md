## [W-76] 사용자별 홈 디렉토리 권한 설정

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-76 |
| 항목명 | 사용자별 홈 디렉토리 권한 설정 |
| 위험도 | 중 (Medium) |
| 범주 | 파일 및 디렉토리 관리 |

### 점검 목적
사용자 홈 디렉토리에 다른 사용자의 읽기/쓰기 권한이 부여된 경우, 내부 위협자나 권한 상승 공격자가 다른 사용자의 개인 파일을 열람하거나 변조할 수 있습니다. 각 사용자의 홈 디렉토리는 해당 사용자에게만 접근 권한을 부여해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 각 사용자 홈 디렉토리에 해당 사용자만 접근 권한이 부여된 경우 | 다른 사용자의 개인 파일 무단 열람 및 변조가 차단된다 |
| 취약 | 사용자 홈 디렉토리에 Everyone, Users 그룹 등 과도한 권한이 부여된 경우 | 다른 사용자의 파일에 무단 접근 및 변조가 가능하다 |
