## [W-08] 하드디스크 기본 공유 제거

디스크 기본 공유 제거

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-08 |
| 항목명 | 하드디스크 기본 공유 제거 |
| 위험도 | 상 (High) |
| 범주 | 공유 및 권한 |

### 점검 목적
Windows는 기본적으로 C$, D$, ADMIN$, IPC$ 등의 관리 목적 기본 공유를 자동 생성합니다. 이 공유들은 관리자 자격증명을 가진 공격자가 원격에서 파일 시스템 전체에 접근하거나 측면 이동(Lateral Movement)에 활용할 수 있는 경로가 됩니다. 업무상 불필요한 기본 공유는 제거해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | C$, D$ 등 드라이브 기본 공유가 비활성화되어 있는 경우 | 원격에서 드라이브 직접 접근 경로가 차단되어 측면 이동 공격 위험이 감소한다 |
| 취약 | C$, D$ 등 기본 공유가 활성화된 상태로 유지되는 경우 | 관리자 자격증명 탈취 시 공격자가 원격에서 전체 파일 시스템에 접근 가능하다 |


### 참고 문서
- 기본 공유: 관리 목적으로 자동 생성되는 공유 드라이브(Administrative share). 이러한 드라이브들은 C$, D$, E$ 등과 같이 이름 뒤에 $가 붙어서 숨겨진 공유로 처리되며, Windows 2000, XP에서는 관리자 ID와 Password를 알고 있으면 네트워크를 통해 이러한 공유 드라이브들에 자유롭게 접근할 수 있음. 그러나 이후 버전 Windows에서는 보안상의 이유로 로컬 시스템의 관리자가 네트워크를 통해 시스템을 관리하지 못하도록 기본적으로 차단됨 [W-17]
