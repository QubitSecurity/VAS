## [W-29] DNS Zone Transfer 설정

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-29 |
| 항목명 | DNS Zone Transfer 설정 |
| 위험도 | 상 (High) |
| 범주 | 네트워크 서비스 |

### 점검 목적
DNS Zone Transfer는 DNS 서버 간 레코드를 동기화하는 기능입니다. 이 기능이 모든 호스트에 허용된 경우 공격자가 Zone Transfer 요청을 통해 해당 도메인의 모든 서브도메인, 서버 IP, 메일 서버 등 내부 인프라 구조 전체를 수집할 수 있습니다. 반드시 허가된 보조 DNS 서버에만 허용해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | Zone Transfer가 특정 보조 DNS 서버 IP에만 허용된 경우 | 허가되지 않은 호스트는 Zone Transfer가 차단되어 내부 DNS 정보가 보호된다 |
| 취약 | Zone Transfer가 모든 호스트에 허용된 경우 | 누구나 전체 DNS 레코드를 수집하여 내부 인프라 구조를 파악할 수 있다 |
