## [W-64] HTTP/FTP/SMTP 배너 차단

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-64 |
| 항목명 | HTTP/FTP/SMTP 배너 차단 |
| 위험도 | 하 (Low) |
| 범주 | 접근 제어 |

### 점검 목적
서비스 배너(Banner)에 소프트웨어 이름, 버전 정보가 포함된 경우 공격자는 해당 버전의 알려진 취약점을 검색하여 표적 공격을 수행할 수 있습니다. HTTP, FTP, SMTP 서비스의 배너 정보를 제거하거나 변경하여 공격자의 사전 정보 수집을 차단해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 서비스 배너에서 버전 정보 및 소프트웨어 명칭이 제거된 경우 | 공격자가 버전 기반 표적 공격을 수행하기 어려워진다 |
| 취약 | 서비스 배너에 소프트웨어명 및 버전 정보가 노출되는 경우 | 노출된 버전 정보를 통해 공격자가 알려진 취약점을 즉시 파악할 수 있다 |
