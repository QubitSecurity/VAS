## [W-62] SNMP Access Control 설정

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-62 |
| 항목명 | SNMP Access Control 설정 |
| 위험도 | 중 (Medium) |
| 범주 | 접근 제어 |

### 점검 목적
SNMP 서비스가 모든 호스트로부터의 요청을 허용하는 경우, 외부 공격자가 커뮤니티 스트링만 알면 어디서든 시스템 정보를 수집하거나 설정을 변경할 수 있습니다. SNMP 접근은 반드시 허가된 관리 서버 IP에만 허용해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | SNMP 접근이 허가된 관리 서버 IP에만 허용된 경우 | 허가되지 않은 호스트의 SNMP 접근이 차단된다 |
| 취약 | SNMP 접근이 모든 호스트에 허용된 경우 | 외부 공격자가 커뮤니티 스트링으로 어디서든 SNMP에 접근할 수 있다 |

### 참고 문서
- 관련 점검항목: W-29(중), W-30(중) [W-31]
