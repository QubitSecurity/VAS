## [W-25] FTP 서비스 구동 점검(재검토)

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-25 |
| 항목명 | FTP 서비스 구동 점검 |
| 위험도 | 상 (High) |
| 범주 | 네트워크 서비스 |

### 점검 목적
FTP는 데이터를 평문으로 전송하는 프로토콜로, 자격증명과 전송 파일이 네트워크 스니핑에 그대로 노출됩니다. 업무상 불필요하게 FTP 서비스가 실행 중인 경우 즉시 중지해야 하며, 필요한 경우 SFTP 또는 FTPS로 대체해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | FTP 서비스가 비활성화되어 있거나 SFTP/FTPS로 대체 운영되는 경우 | 평문 전송에 의한 자격증명 및 데이터 노출 위험이 없다 |
| 취약 | FTP 서비스가 실행 중인 경우 | 자격증명 및 전송 데이터가 네트워크 스니핑에 노출될 수 있다 |
