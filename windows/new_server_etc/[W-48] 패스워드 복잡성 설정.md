## [W-48] 패스워드 복잡성 설정

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-48 |
| 항목명 | 패스워드 복잡성 설정 |
| 위험도 | 중 (Medium) |
| 범주 | 패스워드 정책 |

### 점검 목적
단순한 패스워드는 사전 대입 공격(Dictionary Attack)이나 무차별 대입 공격에 매우 취약합니다. 영문 대소문자, 숫자, 특수문자의 조합을 강제하는 복잡성 정책을 적용하면 패스워드 크래킹에 필요한 시간이 기하급수적으로 증가하여 계정 보안이 강화됩니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 패스워드 복잡성 요구사항이 활성화된 경우 | 영문 대소문자·숫자·특수문자 조합이 강제되어 단순 패스워드 사용이 차단된다 |
| 취약 | 패스워드 복잡성 요구사항이 비활성화된 경우 | 단순 패스워드 설정이 허용되어 사전 대입 공격에 취약해진다 |
