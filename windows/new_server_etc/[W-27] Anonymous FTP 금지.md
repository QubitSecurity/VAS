## [W-27] Anonymous FTP 금지

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-27 |
| 항목명 | Anonymous FTP 금지 |
| 위험도 | 상 (High) |
| 범주 | 네트워크 서비스 |

### 점검 목적
Anonymous FTP는 인증 없이 누구나 FTP 서버에 접속할 수 있도록 허용하는 설정입니다. 활성화된 경우 불특정 외부 공격자가 서버 파일을 열람하거나 악성 파일을 업로드하는 데 악용될 수 있습니다. FTP 서비스 운영 시 반드시 익명 접속을 차단해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | Anonymous FTP 접속이 비활성화된 경우 | 인증 없는 FTP 접근이 차단되어 무단 파일 열람 및 업로드가 불가능하다 |
| 취약 | Anonymous FTP 접속이 허용된 경우 | 인증 없이 누구나 FTP에 접근하여 파일 열람 및 업로드가 가능하다 |
