#### `W-50` `중` 패스워드 최대 사용 기간

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-50 |
| 항목명 | 패스워드 최대 사용 기간 |
| 위험도 | 중 (Medium) |
| 범주 | 패스워드 정책 |

### 점검 목적
패스워드를 영구적으로 사용하는 경우, 패스워드가 유출되더라도 장기간 탈취된 자격증명이 유효한 상태로 유지됩니다. 주기적인 패스워드 변경을 강제하면 탈취된 자격증명의 유효 기간을 제한하여 피해를 최소화할 수 있습니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 패스워드 최대 사용 기간이 90일 이하로 설정된 경우 | 주기적 패스워드 변경으로 탈취된 자격증명의 유효 기간이 제한된다 |
| 취약 | 패스워드 최대 사용 기간이 설정되지 않았거나 90일을 초과한 경우 | 탈취된 자격증명이 장기간 유효하여 지속적인 무단 접근이 가능하다 |
