#### `W-77` `중` LAN Manager 인증 수준

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-77 |
| 항목명 | LAN Manager 인증 수준 |
| 위험도 | 중 (Medium) |
| 범주 | 네트워크 인증 |

### 점검 목적
LAN Manager(LM) 인증은 구형 Windows에서 사용되던 약한 해시 알고리즘입니다. LM 인증을 허용한 경우 패스워드 해시가 약하게 생성되어 오프라인 크래킹 공격에 취약해집니다. NTLMv2 이상만 허용하여 보안성을 강화해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | LAN Manager 인증이 비활성화되고 NTLMv2 이상만 사용되는 경우 | 약한 LM 해시 생성이 차단되어 패스워드 크래킹 난이도가 크게 증가한다 |
| 취약 | LAN Manager 인증이 허용되는 경우 | LM 해시를 통해 패스워드가 쉽게 크래킹될 수 있다 |

### 참고 문서
- LAN Manager는 네트워크를 통한 파일 및 프린터 공유 등과 같은 작업 시 인증을 담당. NTLMv2는 Windows 2000, 2003, XP 이상에서 지원되며, Windows 98, NT 버전과 통신하면 패치를 설치해야 함 [W-59]
