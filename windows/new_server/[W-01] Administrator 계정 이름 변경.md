#### `W-01` `상` Administrator 계정 이름 변경 또는 보안성 강화

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-01 |
| 항목명 | Administrator 계정 이름 변경 또는 보안성 강화 |
| 위험도 | 상 (High) |
| 범주 | 계정 관리 |

### 점검 목적
Windows 기본 관리자 계정인 `Administrator`는 공격자에게 이미 알려진 계정명입니다. 계정명이 기본값 그대로라면 공격자는 아이디를 알고 있는 상태에서 패스워드만 대입하면 되므로 무차별 대입 공격(Brute Force) 성공 가능성이 크게 높아집니다. 계정 이름을 변경하여 공격자가 유효한 계정명을 추측하기 어렵게 해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | Administrator 계정 이름이 변경되어 있는 경우 | 기본 계정명을 사용하지 않아 계정명 유추를 통한 공격을 방어할 수 있다 |
| 취약 | Administrator 계정 이름이 기본값(Administrator)으로 유지되는 경우 | 공격자가 계정명을 이미 알고 있으므로 패스워드 공격만으로 관리자 권한 획득이 가능하다 |
