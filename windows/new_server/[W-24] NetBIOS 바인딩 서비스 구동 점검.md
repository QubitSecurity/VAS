#### `W-24` `상` NetBIOS 바인딩 서비스 구동 점검

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-24 |
| 항목명 | NetBIOS 바인딩 서비스 구동 점검 |
| 위험도 | 상 (High) |
| 범주 | 네트워크 서비스 |

### 점검 목적
NetBIOS는 구형 Windows 네트워크 환경에서 사용되던 프로토콜로, 현재 인터넷 환경에서는 불필요한 경우가 대부분입니다. NetBIOS가 활성화된 경우 공격자는 NetBIOS 이름 조회를 통해 내부 네트워크 구조, 계정 정보, 공유 자원을 수집할 수 있으며 NBNS 스푸핑 공격의 대상이 될 수 있습니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | NetBIOS over TCP/IP가 비활성화된 경우 | NetBIOS를 통한 정보 수집 및 스푸핑 공격 경로가 차단된다 |
| 취약 | NetBIOS over TCP/IP가 활성화된 경우 | 내부 네트워크 정보 노출 및 스푸핑 공격에 악용될 수 있다 |

### 참고 문서
- NetBIOS(Network Basic Input/Output System)는 별개의 컴퓨터상에 있는 응용프로그램들이 근거리통신망 내에서 서로 통신할 수 있게 해주는 API. 이름 서비스, 세션 서비스, 데이터그램 서비스를 제공하며, 주로 네트워크 장치 간의 통신을 지원 [W-20]
