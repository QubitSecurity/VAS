#### `W-71` `중` 원격에서 이벤트 로그파일 접근 차단

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-71 |
| 항목명 | 원격에서 이벤트 로그 파일 접근 차단 |
| 위험도 | 중 (Medium) |
| 범주 | 로그 및 감사 |

### 점검 목적
원격에서 이벤트 로그에 접근할 수 있는 경우, 공격자가 계정 탈취 후 원격에서 보안 로그를 삭제하거나 변조하여 침해 흔적을 은폐할 수 있습니다. 이벤트 로그 접근 권한을 로컬 관리자에게만 제한하고 원격 접근을 차단해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 원격 이벤트 로그 접근이 차단된 경우 | 계정 탈취 시 공격자가 보안 로그를 원격에서 삭제·변조할 수 없다 |
| 취약 | 원격 이벤트 로그 접근이 허용된 경우 | 공격자가 원격에서 보안 로그를 삭제하여 침해 흔적을 은폐할 수 있다 |

### 참고 문서
- 로그 디렉터리 기본 위치
Ÿ 시스템 로그 디렉터리: %systemroot%\system32\config
Ÿ IIS 로그 디렉터리: %systemroot%\system32\LogFiles [W-43]
