#### `W-45` `상` 디스크 볼륨 암호화 설정

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-45 |
| 항목명 | 디스크 볼륨 암호화 설정 |
| 위험도 | 상 (High) |
| 범주 | 보안 설정 |

### 점검 목적
디스크 볼륨 암호화가 적용되지 않은 경우, 서버 하드웨어를 물리적으로 탈취하거나 디스크를 분리하여 다른 시스템에 연결하면 OS 인증 없이 저장된 모든 데이터에 직접 접근할 수 있습니다. BitLocker 등의 볼륨 암호화를 적용하면 물리적 탈취 시에도 데이터를 보호할 수 있습니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | BitLocker 등 디스크 볼륨 암호화가 활성화된 경우 | 디스크 물리적 탈취 시에도 저장 데이터에 접근할 수 없다 |
| 취약 | 디스크 볼륨 암호화가 적용되지 않은 경우 | 디스크 분리 후 타 시스템 연결만으로 모든 데이터에 접근 가능하다 |
