#### `W-03` `상` 불필요한 계정 제거

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-03 |
| 항목명 | 불필요한 계정 제거 |
| 위험도 | 상 (High) |
| 범주 | 계정 관리 |

### 점검 목적
퇴직자, 장기 미사용, 테스트 목적으로 생성된 계정 등 업무상 불필요한 계정이 시스템에 잔존할 경우, 공격자가 해당 계정을 탈취하여 시스템에 접근할 수 있습니다. 계정 수명 주기를 관리하여 불필요한 계정을 삭제하거나 비활성화해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 모든 계정이 현재 업무상 필요하며 정기적으로 검토된 경우 | 불필요한 계정이 없어 불법 접근 경로가 존재하지 않는다 |
| 취약 | 퇴직자, 장기 미사용, 용도 불명 계정 등이 활성화 상태로 존재하는 경우 | 해당 계정을 통한 불법 접근 및 권한 탈취 위험이 있다 |
