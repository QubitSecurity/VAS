#### `W-31` `상` 최신 서비스팩 적용

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-31 |
| 항목명 | 최신 서비스팩 적용 |
| 위험도 | 상 (High) |
| 범주 | 패치 관리 |

### 점검 목적
서비스팩(Service Pack)은 OS에 누적된 보안 패치, 버그 수정, 성능 개선을 통합한 대규모 업데이트입니다. 최신 서비스팩이 적용되지 않은 경우 공개된 취약점이 패치되지 않은 상태로 남아 공격자가 해당 취약점을 직접 악용할 수 있습니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | OS에 최신 서비스팩이 적용된 경우 | 공개된 주요 취약점이 패치되어 알려진 공격에 대한 방어가 이루어진다 |
| 취약 | 최신 서비스팩이 적용되지 않은 경우 | 패치되지 않은 취약점을 통한 공격에 노출될 수 있다 |

### 참고 문서
- 서비스 팩: 운영체제 응용프로그램의 기능 추가 및 버그나 보안 취약점을 해결한 패치 파일을 단일 묶음으로 배포하는 패키지
- 익스플로잇: 컴퓨터의 소프트웨어나 하드웨어 또는 컴퓨터 관련 전자 제품의 버그, 보안 취약점 등 설계상 결함을 이용해 공격자의 의도된 동작을 수행하도록 만들어진 프로그램 [PC-11]
