#### `W-07` `상` 공유 권한 및 사용자 그룹 설정

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-07 |
| 항목명 | 공유 권한 및 사용자 그룹 설정 |
| 위험도 | 상 (High) |
| 범주 | 공유 및 권한 |

### 점검 목적
네트워크 공유 폴더에 Everyone 그룹이 포함되거나 쓰기 권한이 과도하게 부여된 경우, 인증된 내부 사용자뿐 아니라 인증 우회에 성공한 공격자도 파일을 읽거나 악성코드를 업로드할 수 있습니다. 공유 폴더는 반드시 필요한 사용자와 그룹에만 최소 권한으로 설정해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 공유 폴더에 Everyone 그룹이 없고, 필요한 사용자/그룹에만 최소 권한이 부여된 경우 | 불필요한 접근이 차단되어 정보 유출 및 악성코드 업로드 위험이 감소한다 |
| 취약 | 공유 폴더에 Everyone 그룹이 포함되거나 쓰기 권한이 불필요하게 부여된 경우 | 모든 사용자가 공유 자원에 접근 가능하여 정보 유출 및 시스템 침해 위험이 높아진다 |



### 참고 문서
- "응용프로그램이나 백업 용도로 반드시 공유가 필요한 경우에는 Everyone 권한 대신 특정 관리자 계정이나 서비스 전용 계정을 생성하여 최소 권한으로 할당할 것을 권고함." [W-16]
