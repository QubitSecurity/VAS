#### `W-36` `상` 백신 프로그램 설치

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-36 |
| 항목명 | 백신 프로그램 설치 |
| 위험도 | 상 (High) |
| 범주 | 패치 관리 |

### 점검 목적
백신 프로그램이 설치되지 않은 시스템은 악성코드, 랜섬웨어, 트로이목마 등의 감염에 무방비 상태가 됩니다. 서버 환경에서도 반드시 백신 프로그램을 설치하고 실시간 감시 기능을 활성화하여 악성코드 감염을 사전에 방지해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 백신 프로그램이 설치되고 실시간 감시가 활성화된 경우 | 악성코드 감염 시도를 실시간으로 탐지하고 차단할 수 있다 |
| 취약 | 백신 프로그램이 설치되지 않았거나 실시간 감시가 비활성화된 경우 | 악성코드 감염에 무방비 상태가 되어 시스템 전체가 위협에 노출된다 |

### 참고 문서
- 웜: 악의적인 목적을 가지고 자기 자신을 복제해 전파하며 주로 네트워크 공유 폴더나 메일로 전파됨
- 트로이목마: 고의로 악의적 목적이 있는 파일, 주로 다른 악성 코드나 위장된 프로그램으로 전파되거나 인터넷을 통해 다운로드 됨
- 관련 점검항목 : W-39(상) [W-45]
