#### `W-02` `상` Guest 계정 비활성화


### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-02 |
| 항목명 | Guest 계정 비활성화 |
| 위험도 | 상 (High) |
| 범주 | 계정 관리 |

### 점검 목적
Guest 계정은 인증 없이 또는 최소한의 인증만으로 시스템에 접근할 수 있는 기본 계정입니다. 활성화된 상태로 방치되면 인증 우회를 통한 시스템 접근, 내부 정보 열람 등의 경로가 될 수 있습니다. 불필요한 계정은 반드시 비활성화하여 공격 표면을 줄여야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | Guest 계정이 비활성화(사용 안 함)되어 있는 경우 | 불필요한 접근 경로가 차단되어 계정 기반 공격 위험이 감소한다 |
| 취약 | Guest 계정이 활성화되어 있는 경우 | 인증 없이 또는 낮은 권한으로 시스템 접근이 가능하여 정보 유출 및 권한 상승 공격에 악용될 수 있다 |


### 참고 문서
- Windows Guest 계정은 삭제 불가능한 built-in 계정으로 보안 강화 목적으로 반드시 비활성화 처리해야 함 [W-02]
