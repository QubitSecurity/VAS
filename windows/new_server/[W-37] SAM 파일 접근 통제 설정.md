#### `W-37` `상` SAM 파일 접근 통제 설정

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-37 |
| 항목명 | SAM 파일 접근 통제 설정 |
| 위험도 | 상 (High) |
| 범주 | 보안 설정 |

### 점검 목적
SAM(Security Account Manager) 파일은 Windows 로컬 계정의 패스워드 해시를 저장하는 데이터베이스입니다. 이 파일에 대한 접근이 적절히 통제되지 않으면 공격자가 해시를 추출하여 패스 더 해시(Pass-the-Hash) 또는 오프라인 크래킹 공격을 통해 계정을 탈취할 수 있습니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | SAM 파일에 대한 접근이 SYSTEM 계정으로만 제한된 경우 | 일반 계정 및 공격자가 SAM 파일에 접근하여 해시를 추출할 수 없다 |
| 취약 | SAM 파일에 일반 사용자 또는 Everyone 그룹이 읽기 권한을 가진 경우 | 패스워드 해시 추출을 통한 계정 탈취 공격이 가능하다 |

### 참고 문서
- SAM(Security Account Manager): 사용자와 그룹 계정의 비밀번호를 관리하고, LSA(Local Security Authority)를 통한 인증을 제공함 [W-46]
