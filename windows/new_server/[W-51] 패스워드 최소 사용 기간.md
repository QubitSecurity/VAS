#### `W-51` `중` 패스워드 최소 사용 기간

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-51 |
| 항목명 | 패스워드 최소 사용 기간 |
| 위험도 | 중 (Medium) |
| 범주 | 패스워드 정책 |

### 점검 목적
패스워드 최소 사용 기간이 설정되지 않은 경우, 사용자가 패스워드 변경 직후 즉시 이전 패스워드로 되돌릴 수 있어 패스워드 히스토리(W-55) 정책이 무력화됩니다. 최소 1일 이상의 최소 사용 기간을 설정하여 패스워드 재사용 방지 정책의 실효성을 확보해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 패스워드 최소 사용 기간이 1일 이상으로 설정된 경우 | 패스워드 변경 직후 이전 패스워드로의 즉시 복귀가 차단되어 히스토리 정책이 실효성을 가진다 |
| 취약 | 패스워드 최소 사용 기간이 0일(제한 없음)로 설정된 경우 | 패스워드 히스토리 정책이 무력화되어 동일 패스워드를 반복 사용할 수 있다 |
