#### `W-49` `중` 패스워드 최소 암호 길이

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-49 |
| 항목명 | 패스워드 최소 암호 길이 |
| 위험도 | 중 (Medium) |
| 범주 | 패스워드 정책 |

### 점검 목적
패스워드 길이가 짧을수록 가능한 조합의 수가 줄어들어 무차별 대입 공격에 소요되는 시간이 크게 감소합니다. 최소 8자 이상의 패스워드 길이를 강제하여 공격자가 패스워드를 크래킹하는 데 현실적으로 불가능한 시간이 소요되도록 해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 최소 패스워드 길이가 8자 이상으로 설정된 경우 | 패스워드 조합 수가 충분히 커져 무차별 대입 공격 성공 가능성이 크게 낮아진다 |
| 취약 | 최소 패스워드 길이가 7자 이하이거나 설정되지 않은 경우 | 짧은 패스워드는 무차별 대입 공격에 단시간 내 크래킹될 수 있다 |
