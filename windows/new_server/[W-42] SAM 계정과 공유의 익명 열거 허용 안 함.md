#### `W-42` `상` SAM 계정과 공유의 익명 열거 허용 안함

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-42 |
| 항목명 | SAM 계정과 공유의 익명 열거 허용 안 함 |
| 위험도 | 상 (High) |
| 범주 | 보안 설정 |

### 점검 목적
익명 사용자가 SAM 계정 목록과 공유 폴더를 열거할 수 있는 경우, 공격자는 인증 없이 시스템의 계정 목록과 공유 자원 정보를 수집할 수 있습니다. 이 정보는 이후 무차별 대입 공격이나 표적 공격의 사전 정보 수집(Reconnaissance) 단계에서 활용됩니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | SAM 계정 및 공유의 익명 열거가 차단된 경우 | 인증되지 않은 계정·공유 정보 수집이 불가능하여 표적 공격 가능성이 감소한다 |
| 취약 | SAM 계정 및 공유의 익명 열거가 허용된 경우 | 공격자가 인증 없이 계정 목록과 공유 자원 정보를 수집할 수 있다 |

### 참고 문서
- 방화벽과 라우터에서 135~139(TCP, UDP) Port 차단을 통해 외부로부터의 위협을 차단함
- 네트워크 및 전화 접속 연결 > 로컬 영역 > 등록 정보 > 고급 > 고급 설정 > Microsoft 네트워크 파일 및 프린트 공유를 해제해야 함 [W-51]
