#### `W-54` `중` 익명 SID/이름 변환 허용 해제

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-54 |
| 항목명 | 익명 SID/이름 변환 허용 해제 |
| 위험도 | 중 (Medium) |
| 범주 | 패스워드 정책 |

### 점검 목적
SID(Security Identifier)와 계정명 간의 변환이 익명 사용자에게 허용된 경우, 공격자가 인증 없이 시스템의 계정 SID를 조회하여 계정명을 수집하거나 계정명으로부터 SID를 획득할 수 있습니다. 이는 권한 상승 공격의 사전 정보 수집에 활용됩니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 익명 SID/이름 변환이 허용되지 않는 경우 | 인증되지 않은 사용자의 계정 정보 수집이 차단된다 |
| 취약 | 익명 SID/이름 변환이 허용된 경우 | 인증 없이 계정명과 SID를 상호 조회하여 계정 정보를 수집할 수 있다 |

### 참고 문서
- 해당 정책이 설정될 경우 익명 사용자가 다른 사용자의 SID(보안 식별자) 특성을 요청할 수 있음
- “사용 안 함”으로 정책을 설정할 경우 Windows NT 도메인 환경에서 통신 불가능하게 될 수 있음 [W-54]
