#### `W-75` `하` 경고 메시지 설정

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-75 |
| 항목명 | 경고 메시지 설정 |
| 위험도 | 하 (Low) |
| 범주 | 접근 제어 |

### 점검 목적
로그인 시 법적 경고 메시지(Banner)가 표시되지 않으면, 시스템 접근 시 법적 책임 고지와 사용 정책 동의가 이루어지지 않습니다. 로그인 전 적절한 법적 경고 메시지를 설정하여 시스템 접근자의 법적 책임을 명확히 하고, 공격자의 접근을 저지하는 심리적 효과도 기대할 수 있습니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 로그인 시 법적 경고 메시지가 표시되도록 설정된 경우 | 접근자의 법적 책임이 명확히 고지되어 법적 보호가 강화된다 |
| 취약 | 로그인 시 법적 경고 메시지가 표시되지 않는 경우 | 접근자의 법적 책임 고지가 이루어지지 않아 법적 보호가 불충분하다 |
