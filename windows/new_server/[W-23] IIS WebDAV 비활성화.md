## [W-23] IIS WebDAV 비활성화

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-23 |
| 항목명 | IIS WebDAV 비활성화 |
| 위험도 | 상 (High) |
| 범주 | IIS 관리 |

### 점검 목적
WebDAV(Web Distributed Authoring and Versioning)는 HTTP를 통해 원격에서 파일을 생성, 수정, 삭제할 수 있는 프로토콜입니다. 업무상 불필요하게 활성화된 경우 공격자가 WebDAV를 통해 악성 파일을 업로드하거나 서버의 파일 시스템을 원격에서 조작할 수 있습니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | WebDAV가 비활성화된 경우 | HTTP를 통한 원격 파일 조작 경로가 차단된다 |
| 취약 | WebDAV가 활성화된 경우 | 원격에서 HTTP를 통해 파일 업로드 및 서버 파일 시스템 조작이 가능하다 |
