#### `W-38` `상` 화면보호기 설정

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-38 |
| 항목명 | 화면 보호기 설정 |
| 위험도 | 상 (High) |
| 범주 | 보안 설정 |

### 점검 목적
일정 시간 동안 사용자 입력이 없을 때 화면 보호기가 활성화되지 않거나 재개 시 암호를 요구하지 않는 경우, 자리를 비운 사용자의 세션을 물리적으로 접근한 공격자가 그대로 사용할 수 있습니다. 유휴 세션 잠금은 내부자 위협 및 물리적 접근 공격을 방어하는 기본 통제입니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 화면 보호기가 활성화되고 재개 시 암호 입력이 요구되며 대기 시간이 10분 이하인 경우 | 자리를 비운 사이 무단 세션 접근이 차단된다 |
| 취약 | 화면 보호기가 비활성화되거나 재개 시 암호를 요구하지 않는 경우 | 물리적 접근자가 로그인된 세션을 그대로 사용할 수 있다 |
