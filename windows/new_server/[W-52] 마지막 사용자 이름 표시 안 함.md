#### `W-52` `중` 마지막 사용자 이름 표시 안함

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-52 |
| 항목명 | 마지막 사용자 이름 표시 안 함 |
| 위험도 | 중 (Medium) |
| 범주 | 패스워드 정책 |

### 점검 목적
로그인 화면에 이전에 로그인한 사용자 계정명이 표시되는 경우, 공격자는 화면만 보고도 유효한 계정명을 획득할 수 있습니다. 이는 이후 패스워드 대입 공격의 사전 정보로 직접 활용될 수 있으므로 마지막 로그인 사용자 이름 표시를 비활성화해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 로그인 화면에 마지막 사용자 이름이 표시되지 않는 경우 | 계정명이 노출되지 않아 공격자의 정보 수집이 차단된다 |
| 취약 | 로그인 화면에 마지막 사용자 이름이 표시되는 경우 | 계정명이 노출되어 패스워드 대입 공격의 사전 정보로 활용될 수 있다 |

### 참고 문서
- Windows 로그인 화면에 마지막 로그온한 사용자 이름이 표시될 경우 주로 콘솔 사용자 및 터미널 서비스 이용자에게 시스템에 존재하는 사용자 계정 정보를 노출함 [W-10]
