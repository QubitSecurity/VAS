#### `W-39` `상` 로그온 하지 않고 시스템 종료 허용 해제

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-39 |
| 항목명 | 로그온 하지 않고 시스템 종료 허용 해제 |
| 위험도 | 상 (High) |
| 범주 | 보안 설정 |

### 점검 목적
로그온 없이 시스템 종료가 허용된 경우, 인증되지 않은 사용자가 물리적으로 접근하거나 로그인 화면에서 서버를 재부팅하여 서비스 중단(DoS)을 유발할 수 있습니다. 시스템 종료 권한은 반드시 인증된 사용자에게만 부여해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 로그온 없이 시스템 종료가 허용되지 않는 경우 | 인증되지 않은 사용자에 의한 서비스 중단 위험이 차단된다 |
| 취약 | 로그온 화면에서 종료 버튼이 활성화된 경우 | 인증 없이 서버를 종료하여 서비스 중단을 유발할 수 있다 |
