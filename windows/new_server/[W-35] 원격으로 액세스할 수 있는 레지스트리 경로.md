#### `W-35` `상` 원격으로 액세스 할 수 있는 레지스트리 경로

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-35 |
| 항목명 | 원격으로 액세스할 수 있는 레지스트리 경로 |
| 위험도 | 상 (High) |
| 범주 | 보안 설정 |

### 점검 목적
Windows는 네트워크를 통한 원격 레지스트리 접근 경로를 기본으로 허용하는 경우가 있습니다. 원격 레지스트리 접근이 허용된 경우 공격자가 네트워크를 통해 시스템 설정, 소프트웨어 정보, 계정 정보 등을 원격에서 직접 열람하거나 변조할 수 있습니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 원격 접근 가능한 레지스트리 경로가 비어 있거나 최소한으로 제한된 경우 | 네트워크를 통한 레지스트리 무단 접근 및 변조 위험이 감소한다 |
| 취약 | 불필요하게 많은 레지스트리 경로가 원격 접근 허용 목록에 포함된 경우 | 공격자가 원격에서 시스템 설정을 열람하거나 변조할 수 있다 |

### 참고 문서
- 레지스트리: Windows를 실행하는데 필요한 모든 환경설정 데이터를 모아 두는 중앙 저장소
- 원격 레지스트리 서비스: 원격지에 있는 컴퓨터를 한곳에서 집중적으로 관리하기 위한 목적으로 원격 컴퓨터의 레지스트리에 접근할 수 있도록 하는 서비스 [W-44]
