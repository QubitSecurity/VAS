#### `W-81` `중` 시작 프로그램 목록 분석

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-81 |
| 항목명 | 시작 프로그램 목록 분석 |
| 위험도 | 중 (Medium) |
| 범주 | 시스템 시작 |

### 점검 목적
시스템 시작 시 실행되는 프로그램(시작 프로그램, 레지스트리 Run 키 등)에 악성코드나 의심스러운 프로그램이 등록되어 있는 경우, 재부팅마다 자동 실행되어 지속적인 시스템 감염 및 권한 상승이 발생할 수 있습니다. 시작 프로그램을 정기적으로 검토해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 모든 시작 프로그램이 업무상 정당한 프로그램만 등록되어 있는 경우 | 악성코드의 자동 실행이 차단되어 지속적 감염 위험이 없다 |
| 취약 | 시작 프로그램에 의심스러운 실행 파일, PowerShell 스크립트, 외부 URL 호출 등이 포함된 경우 | 재부팅 시 자동 실행되는 악성코드로 시스템이 지속적으로 감염될 수 있다 |
