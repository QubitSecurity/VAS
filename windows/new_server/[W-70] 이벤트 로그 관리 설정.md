#### `W-70` `하` 이벤트 로그 관리 설정

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-70 |
| 항목명 | 이벤트 로그 관리 설정 |
| 위험도 | 하 (Low) |
| 범주 | 로그 및 감사 |

### 점검 목적
이벤트 로그 크기가 너무 작거나 로그가 순환되도록 설정된 경우, 중요한 보안 이벤트가 덮어쓰여 기록이 사라질 수 있습니다. 충분한 크기의 로그 저장 공간을 확보하고 로그가 덮어쓰이지 않도록 설정하여 장기적인 감사 추적을 보장해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 이벤트 로그 최대 크기가 충분하고(최소 10MB), 순환 모드가 적절히 설정된 경우 | 중요한 로그가 덮어쓰이지 않고 장기간 보관되어 감사 추적이 가능하다 |
| 취약 | 이벤트 로그 크기가 너무 작거나 무제한으로 설정되어 로그가 과도하게 누적되는 경우 | 로그가 덮어쓰이거나 디스크 공간 고갈로 시스템 오류가 발생할 수 있다 |
