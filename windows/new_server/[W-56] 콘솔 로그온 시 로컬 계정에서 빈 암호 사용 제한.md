## [W-56] 콘솔 로그온 시 로컬 계정에서 빈 암호 사용 제한

### 개요
| 항목 | 내용 |
|------|------|
| 점검 코드 | W-56 |
| 항목명 | 콘솔 로그온 시 로컬 계정에서 빈 암호 사용 제한 |
| 위험도 | 중 (Medium) |
| 범주 | 패스워드 정책 |

### 점검 목적
패스워드가 설정되지 않은 빈 암호 계정이 콘솔 외 원격 접속에도 허용될 경우, 공격자가 원격에서 해당 계정으로 패스워드 없이 접근할 수 있습니다. 빈 암호 계정은 반드시 콘솔 로컬 접근만 허용하고 네트워크 접근을 차단해야 합니다.

### 점검 기준
| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 빈 암호 계정의 콘솔 로그온만 허용하고 원격 접근은 차단된 경우 | 빈 암호 계정을 통한 원격 무단 접근이 차단된다 |
| 취약 | 빈 암호 계정의 원격 접근이 허용된 경우 | 패스워드 없이 원격에서 해당 계정으로 시스템에 접근할 수 있다 |
