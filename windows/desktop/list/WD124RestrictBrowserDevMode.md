### **WD124 브라우저 개발자 모드 및 확장 프로그램 통제**

**WD124 (WD124RestrictBrowserDevMode)** 항목은 **"웹 브라우저의 개발자 도구 및 비공식 확장 프로그램 사용 제한"**을 의미합니다.

한마디로 요약하면 **"보안 우회나 데이터 추출에 악용될 수 있는 브라우저 개발자 도구(F12)를 일반 사용자가 쓰지 못하게 막고, 검증되지 않은 확장 프로그램 설치를 통제하라"**는 뜻입니다.

---

### 1. 핵심 의미: "브라우저 내 보안 통제 우회 차단"

최근 업무의 상당 부분이 웹 브라우저(Edge, Chrome 등)에서 이루어짐에 따라, 브라우저 자체의 보안 설정을 강화하는 것이 중요해졌습니다.

* **취약한 상태 (제한 없음):**
* 일반 사용자가 **개발자 도구(F12)**를 열어 웹페이지의 소스 코드를 수정하거나, 읽기 전용으로 설정된 데이터를 임의로 조작할 수 있습니다.
* 보안 정책을 우회하는 확장 프로그램(VPN, 우회 프록시 등)을 설치하여 사내 네트워크 보안망을 무력화할 수 있습니다.


* **안전한 상태 (정책 적용):**
* 개발 업무가 필요 없는 일반 직군의 PC에서는 개발자 도구 실행이 차단됩니다.
* 기업이 승인한 공식 확장 프로그램 외에는 설치가 거부되도록 화이트리스트 정책이 적용됩니다.



---

### 2. 점검 목적 (Why?)

"브라우저 도구 좀 쓴다고 보안이 뚫리나요?"

개발자 도구는 숙련된 사용자나 해커에게 웹 앱의 보안 허점을 찾는 강력한 무기가 될 수 있기 때문입니다.

1. **웹 취약점 악용 방지:**
* 스크립트를 조작하여 마우스 우클릭 금지, 드래그 금지 등의 **정보 유출 방지 기능을 무력화**하는 것을 막습니다.


2. **악성 확장 프로그램 차단:**
* 브라우저의 트래픽을 가로채거나 암호를 탈취하는 악성 확장 프로그램(Malicious Extensions) 설치를 원천 차단합니다.


3. **내부 정보 보호:**
* 개발자 도구의 'Network' 탭 등을 통해 서버와 주고받는 API 호출 구조나 중요 파라미터가 외부로 노출되는 것을 최소화합니다.



---

### 3. 점검 기준 (Pass/Fail)

보안 가이드라인에 따른 판단 기준입니다.

* **양호 (Safe):**
* 그룹 정책(GPO)을 통해 브라우저의 "개발자 도구 사용 가능 여부"가 **"사용 안 함"** 또는 **"차단"**으로 설정된 경우


* **취약 (Vulnerable):**
* 별도의 제한 없이 누구나 개발자 도구를 실행하거나 외부 확장 프로그램을 자유롭게 추가할 수 있는 경우



---

### 4. 설정 확인 및 조치 방법 (Chrome/Edge 기준)

이 설정은 AD 환경의 그룹 정책이나 로컬 그룹 정책 편집기를 통해 관리합니다.

1. **실행:** `Win + R` > `gpedit.msc` 입력
2. **경로 이동:**
* **Edge:** `컴퓨터 구성` > `관리 템플릿` > `Microsoft Edge`
* **Chrome:** `컴퓨터 구성` > `관리 템플릿` > `Google Chrome`


3. **정책 찾기 및 조치:**
* **개발자 도구:** **"개발자 도구 사용 가능 여부 제어(DeveloperToolsAvailability)"** 항목을 **[사용(Enabled)]**으로 설정하고, 옵션 값을 **"개발자 도구 사용 불허(2)"**로 선택합니다.
* **확장 프로그램:** **"설치할 수 없는 확장 프로그램 제어"** 항목에 `*`(모두)를 입력하여 전체 차단한 뒤, 필수 프로그램만 허용 리스트에 등록합니다.



---

### 💡 참고: 개발자 직군은 예외?

모든 직원의 개발자 도구를 막으면 업무가 마비될 수 있습니다. 따라서 **조직 단위(OU)**별로 그룹 정책을 분리하여, 개발팀에게는 허용하되 일반 행정/영업 직군에게는 차단하는 유연한 운영이 필요합니다.

### 요약

> **WD124 항목은 "브라우저를 단순한 웹 서핑 도구로 한정시키고, 시스템 내부 구조를 들여다볼 수 있는 도구를 일반 사용자로부터 격리하라"는 의미입니다.**
