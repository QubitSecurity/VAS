## D-10 카메라 및 마이크 접근 제어 활성화

D-10 항목은 **"앱의 카메라 및 마이크 액세스 권한 관리"**를 의미합니다. 사생활 침해 및 도청/감시를 방지하기 위해, 앱이 하드웨어에 접근할 때 반드시 시스템의 승인을 거치고 사용자가 이를 실시간으로 인지할 수 있도록 강제하는 보안 설정입니다.

### 1. 핵심 의미: "사용자 중심의 하드웨어 통제"

SmartScreen이 파일의 안전성을 검사한다면, 이 정책은 하드웨어 자원의 무단 사용을 방지하는 방어막입니다.

* **취약한 상태 (비활성화):** 앱이 백그라운드에서 몰래 카메라/마이크를 작동시켜도 사용자가 알 수 없습니다.
* **안전한 상태 (권한 기반 활성화):** OS가 중간에서 접근을 중재하며, 사용자가 명시적으로 허용한 앱만 리소스를 사용할 수 있습니다.

### 2. 사용자 UI에서의 작동 방식

정책이 적용되면 사용자는 다음과 같은 직관적인 보안 경험을 갖게 됩니다.

* **권한 요청 팝업:** 앱 실행 시 "이 앱이 카메라/마이크를 사용하려고 합니다. 허용하시겠습니까?"라는 팝업이 발생합니다.
* **설정 앱 내 제어:** 윈도우 설정 > 개인 정보 항목에서 앱별 ON/OFF 스위치를 직접 조절할 수 있습니다.
* **트레이 아이콘 알림:** 사용 중일 때 작업 표시줄에 아이콘이 나타나며, 마우스를 올리면 어떤 앱이 사용 중인지 실시간으로 확인 가능합니다.

### 3. 점검 기준 (Pass/Fail)

* **양호 (Safe):** 액세스 설정이 **"사용"**으로 되어 있고, 사용자가 각 앱의 권한을 개별적으로 설정할 수 있도록 허용된 경우.
* **취약 (Vulnerable):** 액세스 설정이 완전히 **"꺼짐"**으로 되어 업무 앱 사용이 불가하거나, 반대로 모든 앱이 제한 없이 접근할 수 있는 경우.

### 4. 설정 방법 (그룹 정책 및 레지스트리)

운영 체제 수준에서 강제 적용하기 위해 아래의 방법을 사용합니다.

#### A. 그룹 정책 편집기 (gpedit.msc)

* **경로:** 컴퓨터 구성 > 관리 템플릿 > Windows 구성 요소 > 앱 개인 정보 보호
* **정책명:** 1. `Windows 앱에서 카메라에 액세스할 수 있도록 허용`
2. `Windows 앱에서 마이크에 액세스할 수 있도록 허용`
* **설정:** **[사용(Enabled)]** 선택 후, 하단 옵션에서 **"사용자가 제어할 수 있음"** 선택.

#### B. 레지스트리 편집기 (regedit)

그룹 정책을 사용할 수 없는 환경이거나 스크립트로 배포할 경우 아래 경로를 수정합니다.

| 구분 | 레지스트리 경로 | 값 이름 (Value Name) | 설정 값 (Value Data) |
| --- | --- | --- | --- |
| **카메라** | `HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\AppPrivacy` | `LetAppsAccessCamera` | **1** (사용자 제어) |
| **마이크** | `HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\AppPrivacy` | `LetAppsAccessMicrophone` | **1** (사용자 제어) |

> [!NOTE]
> **값 데이터(Value Data) 상세 설명:**
> * **0:** 모든 앱에 대해 강제 허용 (보안 취약)
> * **1:** 사용자가 직접 설정 앱에서 제어 가능 (**권장**)
> * **2:** 모든 앱에 대해 강제 거부 (업무 불편 발생 가능)

### 5. 요약 및 서버 운용 팁

* **서버 환경:** 서버는 하드웨어 직접 사용이 드물지만, 관리용 세션에서 발생할 수 있는 보안 사고를 막기 위해 표준 정책을 유지하는 것이 좋습니다.
* **오탐 대응:** 사내 보안 프로그램이 하드웨어를 제어해야 할 경우, 전체 기능을 끄기보다는 해당 앱만 레지스트리 하위 항목에서 예외(AllowList) 처리를 하는 것이 올바른 방향입니다.
