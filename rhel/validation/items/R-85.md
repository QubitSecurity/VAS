## [R-85] 자동 시작 프로그램 목록 점검

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | R-85 |
| 항목명 | 자동 시작 프로그램 목록 점검 |
| 위험도 | 중 (Medium) |
| 범주 | 보안 관리 |

### 점검 목적

시스템 부팅 시 자동으로 시작되는 서비스 중 불필요하거나 의심스러운 항목이 있으면 백도어, 악성코드 등의 지속성(Persistence) 메커니즘으로 악용될 수 있습니다. `systemctl` 활성화 목록을 주기적으로 검토해야 합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 자동 시작 서비스 중 불필요하거나 의심스러운 항목이 없는 경우 | 허가된 서비스만 자동 시작 |
| 취약 | 운영상 불필요하거나 출처 불명의 서비스가 자동 시작으로 등록된 경우 | 백도어, 악성코드 지속성 유지 가능 |

## 베이스라인 기준

**안전(양호)**

- 활성화 서비스 목록이 업무 필수 서비스만으로 구성
- 미확인 서비스 없음
- `/etc/rc.local`에 불명 명령 없음

**취약**

- 불명 서비스가 `enabled` 상태로 등록
- `/etc/rc.local`에 출처 불명 스크립트 실행 명령 존재
