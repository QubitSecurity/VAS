## [R-12] 계정이 존재하지 않는 GID 금지

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | R-12 |
| 항목명 | 계정이 존재하지 않는 GID 금지 |
| 위험도 | 중 (Medium) |
| 범주 | 계정 관리 |

### 점검 목적

`/etc/passwd`에 기록된 사용자의 GID가 `/etc/group`에 정의되어 있지 않을 경우, 그룹 기반 접근 제어가 예측 불가능하게 동작할 수 있습니다. 모든 계정의 GID는 `/etc/group`에 정의된 유효한 그룹이어야 합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | /etc/passwd의 모든 GID가 /etc/group에 정의된 경우 | 그룹 일관성 유지 |
| 취약 | /etc/passwd의 GID 중 /etc/group에 존재하지 않는 항목이 있는 경우 | 그룹 기반 권한 제어 오작동 가능 |

## 베이스라인 기준

**안전(양호)**

- `pwck -r`, `grpck -r` 오류 없음
- 모든 `/etc/passwd` GID가 `/etc/group`에 존재

**취약**

- 고아 GID(orphan GID) 존재
