## [R-42] tftp, talk 서비스 비활성화

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | R-42 |
| 항목명 | tftp, talk 서비스 비활성화 |

| 범주 | 서비스 관리 |

### 점검 목적

TFTP는 인증 기능이 없어 누구나 파일을 다운로드/업로드할 수 있으며, 네트워크 부팅 이외의 목적으로 활성화될 경우 보안 위협이 됩니다. talk/ntalk 역시 불필요한 서비스로 비활성화해야 합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | tftp, talk, ntalk 서비스가 비활성화된 경우 | 인증 없는 파일 전송, 메시징 차단 |
| 취약 | tftp, talk, ntalk 서비스가 활성화된 경우 | 인증 없이 파일 접근, 불필요 서비스 노출 |

## 베이스라인 기준

**안전(양호)**

- tftp, talk, ntalk 모두 `disable = yes` 또는 미설치

**취약**

- 서비스 활성화 상태
