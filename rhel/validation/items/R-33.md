## [R-33] Anonymous FTP 비활성화

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | R-33 |
| 항목명 | Anonymous FTP 비활성화 |
| 위험도 | 상 (High) |
| 범주 | 서비스 관리 |

### 점검 목적

Anonymous FTP를 허용하면 인증 없이 모든 사용자가 FTP 서버에 접속할 수 있습니다. 이를 통해 민감한 파일 다운로드, 악성 파일 업로드, 서버를 불법 콘텐츠 배포지로 악용하는 등의 위험이 있습니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | Anonymous FTP 접속이 차단된 경우 | 인증된 사용자만 FTP 접속 가능 |
| 취약 | Anonymous FTP 접속이 허용된 경우 | 인증 없이 FTP 서버 접근 가능 |

## 베이스라인 기준

**안전(양호)**

- `/etc/vsftpd/vsftpd.conf`: `anonymous_enable=NO`

**취약**

- `anonymous_enable=YES`
