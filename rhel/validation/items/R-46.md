## [R-46] DNS 보안 버전 패치

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | R-46 |
| 항목명 | DNS 보안 버전 패치 |
| 위험도 | 중 (Medium) |
| 범주 | 서비스 관리 |

### 점검 목적

BIND DNS 서버의 구버전에는 원격 코드 실행, DoS 등 심각한 취약점이 다수 존재합니다. DNS 서비스를 운영하는 경우 최신 보안 패치가 적용된 버전을 유지해야 합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | BIND가 최신 보안 패치 버전으로 운영되거나, DNS 서비스 미운영 시 비활성화된 경우 | 알려진 BIND 취약점 없음 |
| 취약 | 취약한 구버전 BIND가 운영 중인 경우 | DNS 서비스 공격을 통한 원격 침해 가능 |

## 베이스라인 기준

**안전(양호)**

- 최신 보안 패치 버전 운영 또는 서비스 비활성화

**취약**

- 알려진 취약점이 있는 구버전
