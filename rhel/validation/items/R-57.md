## [R-57] 로그온 시 경고 메시지 설정

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | R-57 |
| 항목명 | 로그온 시 경고 메시지 설정 |
| 위험도 | 중 (Medium) |
| 범주 | 보안 관리 |

### 점검 목적

로그인 시 법적 경고 문구를 표시함으로써 무단 접근에 대한 법적 제재 근거를 마련하고, 인가되지 않은 접근 시도를 억제할 수 있습니다. 또한 버전 정보가 포함된 기본 배너는 시스템 정보를 노출하므로 제거해야 합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 적절한 법적 경고 메시지가 설정된 경우 | 무단 접근 경고, 시스템 정보 미노출 |
| 취약 | 경고 메시지가 없거나 시스템 버전 정보가 노출되는 경우 | 시스템 정보 노출, 법적 근거 부재 |

## 베이스라인 기준

**안전(양호)**

- `/etc/issue`, `/etc/issue.net`에 OS 버전 정보 없음, 경고 메시지 설정
- `/etc/motd`에 법적 경고문 설정
- SSH `Banner /etc/issue.net` 설정

**취약**

- 기본 배너(`Red Hat Enterprise Linux x.x (Ootpa)` 등) 버전 노출
