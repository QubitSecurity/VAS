## [R-56] 최신 보안 패치 및 벤더 권고사항 적용

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | R-56 |
| 항목명 | 최신 보안 패치 및 벤더 권고사항 적용 |
| 위험도 | 상 (High) |
| 범주 | 보안 관리 |

### 점검 목적

알려진 취약점에 대한 보안 패치가 적용되지 않은 시스템은 공개된 익스플로잇 코드를 통해 손쉽게 공격받을 수 있습니다. 정기적인 보안 패치 적용으로 알려진 취약점을 제거해야 합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 최신 보안 패치가 정기적으로 적용되고 있는 경우 | CVE 기반 알려진 취약점 없음 |
| 취약 | 보안 패치 미적용으로 알려진 취약점이 존재하는 경우 | 공개된 익스플로잇으로 즉시 공격 가능 |

## 베이스라인 기준

**안전(양호)**

- 보안 업데이트 정기 적용 이력 존재
- `yum check-update --security` 결과 없음(최신 상태)

**취약**

- 장기간 패치 미적용
- `security` 분류 패키지 업데이트 다수 존재
