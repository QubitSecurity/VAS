## [R-61] SNMP 서비스 구동 점검

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | R-61 |
| 항목명 | SNMP 서비스 구동 점검 |
| 위험도 | 중 (Medium) |
| 범주 | 보안 관리 |

### 점검 목적

SNMP 서비스가 불필요하게 활성화되어 있으면 네트워크 장비 및 서버 정보가 외부에 노출됩니다. 특히 SNMP v1/v2는 평문 커뮤니티 스트링을 사용하므로 도청에 취약합니다. 미사용 시 비활성화하거나 SNMPv3로 업그레이드해야 합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | SNMP 서비스가 비활성화되거나 SNMPv3를 사용하는 경우 | 불필요한 시스템 정보 노출 차단 또는 암호화 전송 |
| 취약 | SNMP v1/v2가 기본 커뮤니티 스트링으로 활성화된 경우 | 시스템 정보 노출, 도청 취약 |

## 베이스라인 기준

**안전(양호)**

- `snmpd` 서비스 비활성화 또는 SNMPv3 사용
- 기본 커뮤니티 스트링(`public`, `private`) 미사용

**취약**

- SNMPv1/v2 활성화 + 기본 커뮤니티 스트링 사용
