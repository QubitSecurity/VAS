## [R-15] 홈 디렉토리로 PATH 디렉토리 지정 금지

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | R-15 |
| 항목명 | 홈 디렉토리로 PATH 디렉토리 지정 금지 |
| 위험도 | 중 (Medium) |
| 범주 | 계정 관리 |

### 점검 목적

PATH 환경변수에 현재 디렉토리(`.`)나 홈 디렉토리가 포함된 경우, 공격자가 악성 실행 파일을 해당 경로에 위치시켜 관리자가 의도하지 않게 실행하도록 유도할 수 있습니다(Trojan horse 공격). PATH에서 `.` 및 홈 디렉토리를 제거해야 합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | PATH 환경변수에 현재 디렉토리(.) 또는 홈 디렉토리가 포함되지 않은 경우 | 신뢰된 시스템 경로만 PATH에 포함 |
| 취약 | PATH 환경변수에 '.' 또는 홈 디렉토리가 포함된 경우 | 악성 실행 파일 우선 실행 가능(Trojan horse) |

## 베이스라인 기준

**안전(양호)**

- `PATH`에 `.` 또는 빈 문자열(`::`), `$HOME` 미포함

**취약**

- `PATH=.:/usr/local/bin:...` 등 현재 디렉토리 포함
