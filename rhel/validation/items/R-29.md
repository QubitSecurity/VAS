## [R-29] /dev에 존재하지 않는 device 파일 점검

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | R-29 |
| 항목명 | /dev에 존재하지 않는 device 파일 점검 |
| 위험도 | 하 (Low) |
| 범주 | 파일 및 디렉토리 관리 |

### 점검 목적

`/dev` 디렉토리에 정상적인 장치 파일이 아닌 일반 파일이나 숨겨진 파일이 존재할 경우, 이는 루트킷이나 백도어 설치의 흔적일 수 있습니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | /dev 디렉토리에 장치 파일(block, character) 외 일반 파일이 없는 경우 | 정상적인 장치 파일만 존재 |
| 취약 | /dev 디렉토리에 장치 파일이 아닌 일반 파일이 존재하는 경우 | 루트킷, 백도어 의심 |

## 베이스라인 기준

**안전(양호)**

- `/dev`에 일반 파일(`-f`) 없음

**취약**

- `/dev`에 일반 파일 또는 숨김 파일 존재
