## [R-23] /etc/(x)inetd.conf 파일 소유자 및 권한 설정

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | R-23 |
| 항목명 | /etc/(x)inetd.conf 파일 소유자 및 권한 설정 |
| 위험도 | 상 (High) |
| 범주 | 파일 및 디렉토리 관리 |

### 점검 목적

`/etc/inetd.conf` 또는 `/etc/xinetd.conf`는 인터넷 슈퍼 데몬의 서비스 설정 파일입니다. 일반 사용자가 이 파일을 수정할 경우 불필요한 서비스를 활성화하거나 악성 서비스를 추가할 수 있습니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 파일 소유자가 root이고 권한이 600 이하인 경우 | root만 읽기/쓰기 가능 |
| 취약 | 소유자가 root가 아니거나 권한이 600을 초과하는 경우 | 일반 사용자가 서비스 설정 변경 가능 |

## 베이스라인 기준

**안전(양호)**

- `/etc/xinetd.conf` 소유자: `root`, 권한: `600`

**취약**

- other 또는 group write 권한 존재
