## [U-45] 일반 사용자의 Sendmail 실행 방지

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | U-45 |
| 항목명 | 일반 사용자의 Sendmail 실행 방지 |
| 위험도 | 상 (High) |
| 범주 | 서비스 관리 |

### 점검 목적

일반 사용자가 Sendmail을 직접 실행할 수 있으면 메일 스풀 파일을 통한 로컬 권한 상승 취약점이 발생할 수 있습니다. Sendmail 바이너리에 대한 일반 사용자의 실행 권한을 제한해야 합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | Sendmail 바이너리가 root 소유이며 일반 사용자 실행 권한이 없는 경우 | 일반 사용자의 Sendmail 직접 실행 불가 |
| 취약 | 일반 사용자가 Sendmail 바이너리를 실행할 수 있는 경우 | 로컬 권한 상승 취약점 악용 가능 |

## 베이스라인 기준

**안전(양호)**

- `/usr/sbin/sendmail` 소유자 `root`, 권한 `755` 이하
- 불필요한 SUID 없음

**취약**

- 일반 사용자 쓰기 권한 존재
