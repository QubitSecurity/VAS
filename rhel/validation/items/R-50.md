## [R-50] Apache 웹 프로세스 권한 제한

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | R-50 |
| 항목명 | Apache 웹 프로세스 권한 제한 |
| 위험도 | 상 (High) |
| 범주 | 서비스 관리 |

### 점검 목적

Apache가 root 권한으로 실행되면, 웹 취약점을 통해 공격자가 시스템 전체를 장악할 수 있습니다. Apache는 별도의 비권한 계정(예: `apache`)으로 실행되어야 합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | Apache가 root 이외의 전용 계정(apache)으로 실행되는 경우 | 웹 취약점 악용 시 피해 범위 제한 |
| 취약 | Apache가 root 권한으로 실행되는 경우 | 웹 취약점 악용 시 시스템 전체 제어 가능 |

## 베이스라인 기준

**안전(양호)**

- `User apache`, `Group apache`
- 프로세스 실행 계정 `apache` (root 아님)

**취약**

- `User root`, `Group root`
