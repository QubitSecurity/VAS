## [R-26] SUID, SGID, Sticky bit 설정 파일 점검

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | R-26 |
| 항목명 | SUID, SGID, Sticky bit 설정 파일 점검 |
| 위험도 | 상 (High) |
| 범주 | 파일 및 디렉토리 관리 |

### 점검 목적

SUID/SGID가 설정된 실행 파일은 실행 시 파일 소유자(보통 root)의 권한으로 동작합니다. 불필요한 SUID/SGID 파일이 많을수록 권한 상승 공격(Privilege Escalation)에 악용될 수 있는 경로가 증가합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | 목적에 맞지 않는 SUID/SGID 파일이 없는 경우 | 시스템 운영에 반드시 필요한 파일에만 SUID/SGID 설정 |
| 취약 | 불필요한 SUID/SGID 파일이 존재하는 경우 | 권한 상승 공격 경로 증가 |

## 베이스라인 기준

**안전(양호)**

- 시스템 운영에 필요한 SUID/SGID 파일만 존재
- 불필요한 SUID/SGID 파일 제거됨

**취약**

- 기본 목록 이외의 SUID/SGID 파일 존재
- 사용자 작성 스크립트에 SUID 설정
