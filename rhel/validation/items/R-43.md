## [R-43] Sendmail 버전 점검

### 개요

| 항목 | 내용 |
|------|------|
| 점검 코드 | R-43 |
| 항목명 | Sendmail 버전 점검 |
| 위험도 | 상 (High) |
| 범주 | 서비스 관리 |

### 점검 목적

구버전 Sendmail에는 원격 코드 실행, 버퍼 오버플로우 등 심각한 취약점이 다수 존재합니다. 메일 서비스를 운영하는 경우 최신 보안 패치가 적용된 버전을 사용해야 합니다.

### 점검 기준

| 구분 | 기준 | 설명 |
|------|------|------|
| 안전 | Sendmail 최신 버전 또는 보안 패치가 적용된 경우, 또는 메일 서비스 미운영 시 비활성화된 경우 | 알려진 Sendmail 취약점 없음 |
| 취약 | Sendmail이 취약한 구버전으로 운영 중인 경우 | 원격 코드 실행, 권한 상승 등 심각한 취약점 존재 |

## 베이스라인 기준

**안전(양호)**

- 최신 보안 패치 적용 버전 운영
- 메일 서비스 불필요 시 비활성화

**취약**

- 알려진 취약점이 있는 구버전 운영
